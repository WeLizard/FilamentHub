# План очистки и наполнения базы данных FilamentHub

> **Статус:** Требует одобрения перед выполнением  
> **Дата:** 20 ноября 2024

## 🎯 Цель

Подготовить базу данных к бета-тестированию:
1. Удалить все тестовые данные
2. Переименовать админа
3. Наполнить реальными российскими брендами и материалами

---

## 📋 Этап 1: Очистка тестовых данных

### 1.1. Удалить тестовые отзывы
```sql
-- Деактивировать все отзывы
UPDATE filament_reviews SET active = false;
-- Или полностью удалить
DELETE FROM filament_reviews;
```

### 1.2. Удалить тестовые пресеты
```sql
-- Сначала удалить связи с сохранёнными пресетами
DELETE FROM user_saved_presets;
-- Потом удалить сами пресеты
DELETE FROM presets;
```

### 1.3. Удалить тестовые материалы
```sql
-- Сначала проверить, что нет связанных отзывов и пресетов
-- Затем удалить филаменты
DELETE FROM filaments;
```

### 1.4. Удалить тестовые бренды
```sql
-- Проверить что нет связанных материалов
-- Оставить только User Materials бренд (id = 1)
DELETE FROM brands WHERE id != 1;
```

### 1.5. Удалить/деактивировать тестовых пользователей
```sql
-- Оставить только админа
-- Деактивировать всех пользователей кроме админа
UPDATE users SET active = false WHERE role != 'admin';
-- Или полностью удалить
DELETE FROM users WHERE role = 'user';
```

### 1.6. Очистить уведомления
```sql
DELETE FROM notifications;
```

### 1.7. Очистить заявки на бренды/принтеры
```sql
DELETE FROM brand_requests;
DELETE FROM printer_requests;
```

## 📋 Этап 2: Переименование админа

### 2.1. Обновить данные админа
```sql
UPDATE users 
SET 
  username = 'lizard_admin',  -- Или другое имя
  email = 'admin@filamenthub.ru',  -- Твой реальный email
  full_name = 'Администратор FilamentHub'
WHERE role = 'admin';
```

### 2.2. Добавить бейдж founder админу
```sql
UPDATE users 
SET badges = '["founder"]'
WHERE role = 'admin';
```

## 📋 Этап 4: Добавление примеров материалов

### 4.1. Для каждого бренда добавить 2-3 популярных материала

Например для REC:
- REC PLA Red (1.75mm, 750g)
- REC PETG Black (1.75mm, 750g)
- REC ABS White (1.75mm, 750g)

### 4.2. Параметры материалов

Примерные значения для PLA:
- Температура экструдера: 190-220°C
- Температура стола: 50-60°C
- Скорость печати: 40-80 mm/s
- Диаметр: 1.75mm или 2.85mm
- Вес катушки: 750g, 1kg

### 4.3. Цвета

Стандартные цвета для начала:
- Black (Чёрный)
- White (Белый)
- Red (Красный)
- Blue (Синий)
- Green (Зелёный)
- Yellow (Жёлтый)
- Orange (Оранжевый)
- Grey (Серый)
- Transparent (Прозрачный)

## ⚠️ Важные замечания

### Что НЕ удалять:
- ❌ Таблицу `material_types` (справочник типов материалов)
- ❌ Таблицу `bad_words` (словарь запрещённых слов)
- ❌ Таблицу `printers` (справочник принтеров)
- ❌ Бренд "User Materials" (id = 1) - для пользовательских материалов
- ❌ Админа

### Последовательность выполнения:
1. ✅ Сначала удалить зависимые данные (отзывы, пресеты)
2. ✅ Потом основные данные (материалы, бренды)
3. ✅ Затем пользователей
4. ✅ В конце - переименовать админа
5. ✅ Наполнить новыми данными

### Бэкап перед очисткой:
```bash
# Создать дамп БД перед очисткой
docker exec filamenthub-postgres pg_dump -U filamenthub filamenthub > backup_before_cleanup.sql
```
