# 🔧 Dev Scripts

Эта папка содержит скрипты для разработки и отладки.

## 📋 Содержимое

### Check скрипты
- `check_*.py` - скрипты для проверки данных, валидации и отладки

### Debug скрипты
- `debug_*.py` - скрипты для отладки и диагностики

## ⚠️ Важно

Эти скрипты **НЕ используются** в продакшене и предназначены только для разработки.

Они могут:
- Использовать устаревшие API
- Содержать временные решения
- Требовать специальной настройки окружения

## 🚀 Использование

```bash
# Пример запуска check скрипта
cd backend
python dev_scripts/check_presets.py

# Пример запуска debug скрипта
python dev_scripts/debug_printer.py
```
