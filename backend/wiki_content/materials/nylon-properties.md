# Nylon (PA, Полиамид) - Полное руководство

## Что такое Nylon?

**Nylon (PA, Polyamide, Полиамид)** — сверхпрочный и гибкий термопластик. Один из самых прочных материалов для FDM-печати. Используется для функциональных деталей требующих высокую прочность и износостойкость.

## Физические свойства

| Свойство | Значение |
|----------|---------|
| **Температура печати** | 245–260°C (оптимально 250–255°C) |
| **Температура стола** | 80–100°C (рекомендуется 90°C) |
| **Температура камеры** | 40–50°C (закрытая камера желательна) |
| **Температура стеклования (Tg)** | 50–60°C |
| **Плотность** | 1.13–1.15 г/см³ |
| **Прочность на растяжение** | 50–70 МПа (очень высокая) |
| **Удлинение при разрыве** | 30–100% (очень гибкий) |
| **Усадка** | 1.0–2.0% (очень высокая!) |

## Преимущества

✅ **Сверхпрочность**
- Очень высокая прочность на растяжение (50–70 МПа)
- Высокая ударная вязкость
- Износостойкость (подшипники, втулки)

✅ **Гибкость**
- Удлинение при разрыве: 30–100%
- Гнётся, не ломается
- Подходит для пружин, защёлок

✅ **Термостойкость**
- Выдерживает до 80–90°C
- Выше чем у PLA/PETG

✅ **Химическая стойкость**
- Устойчив к маслам, бензину, большинству химикатов

## Недостатки

❌ **КРИТИЧЕСКАЯ гигроскопичность**
- **Очень высокая** впитываемость влаги (8–10%)
- Влажный Nylon → пузыри, трещины, расслоение
- **Обязательная сушка перед печатью:** 80°C, 8–12 часов

❌ **Warping**
- Очень высокая усадка (1.0–2.0%)
- Требует закрытую камеру
- Высокая температура стола обязательна

❌ **Сложность печати**
- Высокие температуры (250°C+)
- Требует опыт калибровки
- Абразивный (изнашивает сопло)

❌ **Хранение**
- Обязательно герметичное (с силикагелем)
- Влажный Nylon → испорченный филамент

## Когда использовать Nylon?

### ✅ Идеально подходит для:

1. **Функциональные детали**
   - Подшипники, втулки, скользящие элементы
   - Защёлки, пружины (гибкость)
   - Детали под высокой нагрузкой

2. **Механические узлы**
   - Шестерни, ролики
   - Детали с износом
   - Подвижные механизмы

3. **Прочные крепления**
   - Кронштейны под нагрузкой
   - Защёлки (гнутся, не ломаются)

### ❌ НЕ подходит для:

1. **Новичков**
   - Сложная печать
   - Требует сушилку для филамента

2. **Без сушки**
   - Испорченная печать гарантирована
   - Обязательна сушилка (80°C, 8–12ч)

3. **Детали требующие точности**
   - Высокая усадка → размеры неточные

## Настройки печати (рекомендации)

```
Температура сопла: 255°C (250–260°C диапазон)
Температура стола: 90°C (80–100°C)
Температура камеры: 45°C (40–50°C, закрытая камера)
Скорость печати: 25–40 mm/s (медленно!)
Охлаждение: 0–25% (минимальный обдув)
Ретракт: 5–7 мм
Высота слоя: 0.2 мм (0.15–0.25 мм)
Infill: 30–50%
```

### Критически важно:

1. **СУШКА филамента ОБЯЗАТЕЛЬНА**
   - 80°C, 8–12 часов перед печатью
   - Сразу из сушилки в принтер
   - Влажный Nylon → пузыри, трещины

2. **Закрытая камера**
   - Предотвращает warping
   - Температура камеры: 40–50°C

3. **Температура стола высокая**
   - 90°C минимум
   - Brim обязателен

4. **Сопло из стали**
   - Абразивный материал
   - Латунное сопло износится быстро

## Хранение

- **Гигроскопичность:** ⚠️ **ОЧЕНЬ ВЫСОКАЯ (8–10%)**
- **Сушка перед печатью:** **ОБЯЗАТЕЛЬНО: 80°C, 8–12ч**
- **Хранение:** 
  - Герметичный пакет + силикагель
  - 20°C (прохладное место)
  - Вакуумный пакет идеально

### Признаки влаги в Nylon:

- Треск/хруст при печати
- Пузыри в детали
- Расслоение слоёв
- Хрупкие детали

**Решение:** Немедленная сушка 80°C, 12 часов

## Troubleshooting

### Расслоение (delamination)

**Причины:** Влажный филамент, низкая температура

**Решения:**
1. Просушить филамент (80°C, 12ч)
2. Увеличить температуру сопла на 5–10°C
3. Отключить обдув (0%)
4. Закрыть камеру (40–50°C)

### Warping (коробление)

**Решения:**
1. Закрытая камера (40–50°C)
2. Brim или Raft
3. Температура стола 90°C+
4. Медленная печать первого слоя

### Пузыри в детали

**Причина:** Влажный филамент

**Решение:** Просушить филамент перед печатью (80°C, 8–12ч)

## Варианты Nylon

### PA6
- Стандартный Nylon
- Температура печати: 245–260°C

### PA12
- Менее гигроскопичный чем PA6
- Легче печатать
- Дороже

### Nylon с наполнителями
- **Carbon Fiber Nylon:** Повышенная жёсткость
- **Glass Fiber Nylon:** Износостойкость

## Постобработка

1. **Шлифовка:** Легко
2. **Склейка:** Суперклей или эпоксидка
3. **Покраска:** Акриловые краски
