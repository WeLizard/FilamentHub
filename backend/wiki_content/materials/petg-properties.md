# PETG - Полное руководство

## Что такое PETG?

**PETG (Polyethylene Terephthalate Glycol-modified)** — это модифицированный полиэтилентерефталат, который сочетает легкость печати PLA с прочностью и термостойкостью ABS. Один из самых универсальных материалов для FDM-печати.

## Физические свойства

| Свойство | Значение |
|----------|---------|
| **Температура печати** | 220–250°C (оптимально 235–245°C) |
| **Температура стола** | 70–85°C (рекомендуется 80°C) |
| **Температура стеклования (Tg)** | 75–80°C |
| **Плотность** | 1.27 г/см³ |
| **Прочность на растяжение** | 50–55 МПа |
| **Удлинение при разрыве** | 15–30% (гибкий) |
| **Модуль упругости** | 2000–2100 МПа |
| **Усадка** | 0.4–0.8% (средняя) |

## Преимущества

✅ **Прочность и гибкость**
- Высокая ударная вязкость (не ломается при ударе)
- Гнётся перед разрушением (в отличие от PLA)
- Хорошая межслойная адгезия

✅ **Термостойкость**
- Выдерживает до 70–75°C без деформации
- Можно использовать в автомобиле летом
- Подходит для функциональных деталей

✅ **Химическая стойкость**
- Устойчив к большинству бытовых химикатов
- Устойчив к воде, маслам, жирам
- **Пищевой PETG** существует (FDA approved)

✅ **Прозрачность**
- Полупрозрачный (можно сделать прозрачные детали)
- Широкая цветовая палитра
- Глянцевая поверхность

✅ **Баланс печати**
- Проще, чем ABS (не требует закрытой камеры)
- Минимальный warping
- Слабый запах

## Недостатки

❌ **Адгезия к платформе**
- Очень сильно прилипает к стеклу (может вырвать кусок)
- Требует правильная подготовка платформы (клей-карандаш, PEI, BuildTak)
- Не использовать голое стекло!

❌ **Склонность к stringing (волосатость)**
- Филамент тянется между частями модели
- Требует точная настройка ретракта
- Температура критична

❌ **Гигроскопичность**
- Сильно впитывает влагу (как PLA)
- Влажный PETG → пузыри, трещины, ухудшение слойности
- **Обязательная сушка:** 65–70°C, 6–8 часов

❌ **Постобработка**
- Трудно шлифовать (налипает на наждачку)
- Не полируется ацетоном
- Клеить сложнее, чем PLA

❌ **Абразивность**
- Изнашивает латунное сопло (рекомендуется сталь)

## Когда использовать PETG?

### ✅ Идеально подходит для:

1. **Функциональные детали**
   - Крепления, держатели, втулки
   - Детали, работающие под нагрузкой
   - Защёлки, клипсы (гнутся, не ломаются)

2. **Механические детали**
   - Корпуса, чехлы
   - Детали для автомобиля (внутри салона)
   - Детали для улицы (кратковременно)

3. **Пищевой контакт**
   - Контейнеры, держатели для кухни (пищевой PETG!)
   - НО: пористость FDM → бактерии, мыть нельзя в посудомойке

4. **Прозрачные детали**
   - Световоды
   - Декоративные прозрачные элементы

### ❌ НЕ подходит для:

1. **Высокоточные детали**
   - Stringing ухудшает качество мелких элементов
   - Требует тщательная калибровка

2. **Высокотемпературные применения**
   - Выше 70°C → деформация
   - Под капотом авто — нет

3. **Детали с резьбой**
   - Адгезия слишком сильная → резьба может сорваться
   - Лучше PLA или ABS

## Настройки печати (рекомендации)

```
Температура сопла: 240°C (235°C для деталей, 245°C для первого слоя)
Температура стола: 80°C (весь процесс)
Скорость печати: 40–50 mm/s (стены), 60 mm/s (заполнение)
Охлаждение: 30–50% (НЕ 100%! Иначе расслоение)
Ретракт: 5–7 мм (прямой), 7–9 мм (боуден)
Скорость ретракта: 30–40 mm/s (медленнее, чем PLA!)
Z-hop: 0.2–0.4 мм (помогает против stringing)
```

### Подготовка платформы

1. **PEI-поверхность:** идеально, легко снимается
2. **Стекло + клей-карандаш:** тонкий слой (иначе прилипнет навечно)
3. **BuildTak, магнитные платформы:** хорошо
4. **НИКОГДА голое стекло!** — вырвет кусок стекла

## Варианты PETG

### PETG Carbon Fiber (CF)
- Углеволокно для жёсткости
- Матовая поверхность
- Требует сопло из закалённой стали

### PETG Pro / Tough
- Повышенная ударная вязкость
- Лучшая межслойная адгезия

### Transparent PETG
- Максимальная прозрачность
- Требует медленная печать для качества

## Постобработка

1. **Шлифовка:** Плохо (налипает на наждачку), можно влажная шлифовка
2. **Склейка:** Суперклей или эпоксидка
3. **Покраска:** Акриловые краски, грунтовка обязательна
4. **Химическая полировка:** Не работает (не растворяется в ацетоне)
5. **Термообработка:** Не рекомендуется (может деформироваться)

## Troubleshooting

### Stringing (волосатость)
- **Решение:** Снизить температуру (-5°C), увеличить ретракт (+1мм), включить Z-hop

### Плохая адгезия первого слоя
- **Решение:** Увеличить температуру стола до 85°C, добавить клей-карандаш

### Расслоение (layer separation)
- **Решение:** Увеличить охлаждение до 40–50%, поднять температуру на 5°C

### Пористость верхних слоёв
- **Решение:** Снизить охлаждение до 30%, увеличить flow на 2–5%

## Хранение

- **Сухое место:** ≤40% влажности
- **Герметичный контейнер** с силикагелем
- **Сушка перед печатью:** 65–70°C, 6–8 часов (обязательно!)
- **Признаки влаги:** пузыри, треск, хрупкие детали
