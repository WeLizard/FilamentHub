# TPU/TPE (Flexible) - Полное руководство

## Что такое TPU?

**TPU (Thermoplastic Polyurethane, TPE, Flex)** — гибкий эластичный материал для 3D-печати. Может растягиваться и возвращаться в исходную форму. Используется для печати упругих деталей, прокладок, уплотнителей.

## Физические свойства

| Свойство | Значение |
|----------|---------|
| **Температура печати** | 210–230°C (оптимально 220–225°C) |
| **Температура стола** | 50–60°C (опционально) |
| **Shore Hardness** | Shore A 85–95 (мягкий), Shore D 40–50 (жёсткий) |
| **Удлинение при разрыве** | 50–100% (растягивается) |
| **Усадка** | 0.1–0.2% (низкая) |

## Преимущества

✅ **Гибкость и эластичность**
- Растягивается и возвращается в форму
- Не ломается при изгибе
- Подходит для упругих деталей

✅ **Ударопоглощение**
- Амортизация ударов
- Защита хрупких деталей

✅ **Водостойкость**
- Не впитывает воду
- Хорошо для уплотнителей

✅ **Хорошая адгезия**
- Прилипает к большинству поверхностей
- Легко печатать на различных платформах

## Недостатки

❌ **Медленная печать**
- Скорость: 10–30 mm/s (очень медленно!)
- Быстрая печать → пропуски экструзии

❌ **Сложность экструзии**
- Тянется при ретракте
- Требует специальные настройки ретракта
- Может застревать в боуден-трубке

❌ **Stringing (волосатость)**
- Очень склонен к stringing
- Требует точная настройка ретракта

❌ **Требования к принтеру**
- Прямой экструдер предпочтителен
- Боуден может иметь проблемы (застревание)

## Когда использовать TPU?

### ✅ Идеально подходит для:

1. **Упругие детали**
   - Прокладки, уплотнители
   - Амортизаторы, подушки
   - Защитные чехлы

2. **Функциональные элементы**
   - Защёлки (гнутся, не ломаются)
   - Кабельные втулки
   - Держатели (обхватывают предмет)

3. **Прототипирование**
   - Резиновые детали
   - Мягкие игрушки

### ❌ НЕ подходит для:

1. **Жёсткие детали**
   - Нужна жёсткость → использовать PLA/ABS

2. **Высокоскоростная печать**
   - Медленная печать обязательна

3. **Точные размеры**
   - Эластичность → размеры могут плавать

## Настройки печати (рекомендации)

```
Температура сопла: 225°C (220–230°C)
Температура стола: 60°C (50–60°C, опционально)
Скорость печати: 15–25 mm/s (МЕДЛЕННО!)
Охлаждение: 50–100% (после первых слоёв)
Ретракт: МИНИМАЛЬНЫЙ или ВЫКЛЮЧЕН (0–2 мм)
Скорость ретракта: 10–20 mm/s (медленно)
Z-hop: 0.2–0.4 мм
Высота слоя: 0.2–0.3 мм (толще = лучше адгезия)
```

### Критически важно:

1. **МЕДЛЕННАЯ печать**
   - 15–25 mm/s максимум
   - Быстрее → пропуски, плохое качество

2. **Минимальный ретракт**
   - 0–2 мм (или выключен)
   - TPU тянется при ретракте → застревает

3. **Прямой экструдер предпочтителен**
   - Боуден может иметь проблемы
   - Застревание в трубке

4. **Увеличенный flow**
   - 105–110% (компенсация эластичности)

## Troubleshooting

### Пропуски экструзии (under-extrusion)

**Причины:** Слишком быстрая печать, застревание в боуден

**Решения:**
1. Снизить скорость до 15–20 mm/s
2. Увеличить температуру на 5°C
3. Проверить боуден-трубку (застревание)
4. Увеличить flow до 110%

### Stringing (волосатость)

**Причины:** Высокая температура, ретракт

**Решения:**
1. Снизить температуру на 5–10°C
2. Минимальный ретракт (0–1 мм)
3. Включить Z-hop
4. Медленная печать

### Застревание в экструдере

**Решения:**
1. Использовать прямой экструдер (не боуден)
2. Проверить зажим экструдера (не зажат слишком сильно)
3. Температура не слишком высокая (иначе мягкий)

### Плохая адгезия слоёв

**Решения:**
1. Увеличить температуру сопла на 5°C
2. Снизить скорость печати
3. Увеличить высоту слоя до 0.3 мм

## Варианты TPU

### TPU 85A (мягкий)
- Shore A 85
- Очень мягкий, гибкий
- Для уплотнителей, прокладок

### TPU 95A (средний)
- Shore A 95
- Баланс жёсткости и гибкости
- Для большинства применений

### TPU D40/D50 (жёсткий)
- Shore D 40–50
- Жёстче, но всё ещё гибкий
- Для защёлок, держателей

## Постобработка

1. **Шлифовка:** Плохо (мягкий материал)
2. **Склейка:** Суперклей или специальный клей для резины
3. **Резка:** Острый нож или ножницы

## Хранение

- **Гигроскопичность:** Низкая
- **Сушка:** Опционально (60°C, 2ч) если есть признаки влаги
- **Хранение:** Герметичный пакет, 25°C
