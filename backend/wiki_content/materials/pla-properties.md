# PLA (Полилактид) - Полное руководство

## Что такое PLA?

**PLA (Polylactic Acid, полилактид)** — биоразлагаемый термопластик, получаемый из возобновляемых источников (кукуруза, сахарный тростник, картофель). Это самый популярный материал для FDM/FFF 3D-печати, особенно среди новичков.

## Физические свойства

| Свойство | Значение |
|----------|---------|
| **Температура печати** | 170–230°C (оптимально 190–210°C) |
| **Температура стола** | 50–60°C (опционально, можно без подогрева) |
| **Температура плавления** | ~150–160°C |
| **Температура стеклования (Tg)** | 60–65°C |
| **Плотность** | 1.24–1.25 г/см³ |
| **Прочность на растяжение** | 37–50 МПа (зависит от производителя) |
| **Удлинение при разрыве** | 3–6% (хрупкий) |
| **Модуль упругости** | 2500–3000 МПа (жёсткий) |
| **Усадка** | 0.3–0.5% (низкая) |

## Преимущества

✅ **Легкость печати**
- Не требует подогрева стола (но рекомендуется 50°C)
- Низкая усадка → минимальный warping
- Хорошая адгезия к платформе

✅ **Безопасность**
- Практически отсутствует запах
- Не токсичен (при печати ниже 200°C)
- Биоразлагаемый (в промышленных условиях компостирования)

✅ **Эстетика**
- Широкая цветовая палитра
- Глянцевая поверхность
- Хорошее качество нависов (overhang)

✅ **Доступность**
- Самый дешёвый материал ($10–15/кг)
- Доступен везде

## Недостатки

❌ **Низкая термостойкость**
- Начинает деформироваться при 60°C
- Не подходит для деталей, работающих при повышенных температурах
- Солнечный свет → деформация летом в машине

❌ **Хрупкость**
- Ломается при ударе (низкая ударная вязкость)
- Не подходит для функциональных деталей с нагрузкой
- Не гнётся (в отличие от PETG, Nylon)

❌ **Химическая стойкость**
- Растворяется в дихлорметане (CH₂Cl₂), ТГФ
- Разрушается под воздействием сильных щелочей
- Не устойчив к ацетону (но лучше, чем ABS)

❌ **Гигроскопичность**
- Впитывает влагу из воздуха
- Влажный филамент → трещины, пузыри, плохое качество
- **Требует сушки:** 60–80°C в течение 4–8 часов

## Когда использовать PLA?

### ✅ Идеально подходит для:

1. **Прототипирование и визуальные модели**
   - Макеты зданий
   - Демонстрационные образцы
   - Фигурки, декор

2. **Обучение и первая печать**
   - Самый простой материал для новичков
   - Калибровка принтера

3. **Детали без механической нагрузки**
   - Корпуса для электроники (при комнатной температуре)
   - Органайзеры, держатели

4. **Биоразлагаемые изделия**
   - Одноразовая посуда (при использовании пищевого PLA)
   - Экологичные проекты

### ❌ НЕ подходит для:

1. **Функциональные детали**
   - Крепления, подшипники, втулки
   - Детали под нагрузкой/ударом

2. **Высокотемпературные применения**
   - Автомобильные детали (багажник, под капотом)
   - Детали возле нагревателей

3. **Детали на улице**
   - UV-излучение → деградация
   - Температурные циклы → хрупкость

4. **Контакт с агрессивными веществами**
   - Растворители (дихлорметан, ТГФ)
   - Сильные кислоты/щелочи

## Настройки печати (рекомендации)

```
Температура сопла: 200°C (195°C для мостов, 210°C для первого слоя)
Температура стола: 60°C (первый слой), 50°C (остальные)
Скорость печати: 50–60 mm/s (стены), 80 mm/s (заполнение)
Охлаждение: 100% (после 3-5 слоя)
Ретракт: 4–6 мм (прямой экструдер), 6–8 мм (боуден)
Скорость ретракта: 40–50 mm/s
```

## Варианты PLA

### PLA+
- Улучшенная прочность и ударная вязкость
- Меньше хрупкости
- Температура печати: 205–220°C

### PLA Pro (Tough PLA)
- Ещё более прочный (близко к ABS)
- Термостойкость до 70°C
- Температура печати: 210–230°C

### PLA с наполнителями
- **Wood PLA** (древесные волокна): эффект дерева
- **Metal PLA** (металлический порошок): эффект бронзы, меди, латуни
- **Carbon PLA** (углеволокно): жёсткость, матовая поверхность
- **Glow PLA** (люминофор): светится в темноте

## Постобработка

1. **Шлифовка:** Легко, но пыль токсична (используйте маску)
2. **Склейка:** Суперклей (цианоакрилат) или эпоксидная смола
3. **Покраска:** Акриловые краски, грунтовка обязательна
4. **Отжиг (annealing):** 60–70°C в духовке → повышение кристалличности → прочность +25%, но деталь может деформироваться

## Хранение

- **Сухое место:** ≤50% влажности
- **Запечатанный контейнер** с силикагелем
- **Сушка перед печатью:** если филамент хрустит или булькает — сушка 60°C, 4–6 часов
