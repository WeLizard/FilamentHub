# Stringing (Волосатость) - Полное руководство

## Что такое Stringing?

**Stringing (волосатость, паутина)** — это дефект 3D-печати, при котором между частями детали появляются тонкие нити пластика. Сопло оставляет "волоски" при перемещении между участками печати.

![Stringing пример](https://via.placeholder.com/600x300?text=Stringing+Example)

## Как выглядит?

### Лёгкий stringing:
- Редкие тонкие нити (диаметр < 0.1 мм)
- Легко удаляются пальцами/феном
- Не влияет на функциональность

### Средний stringing:
- Множество нитей между элементами
- "Паутина" на детали
- Требует зачистка/обжиг

### Критический stringing:
- Толстые нити (0.3+ мм)
- "Сопли" пластика
- Портит внешний вид и геометрию

## Причины

### 1. Температура слишком высокая

**Что происходит:**
- Пластик слишком жидкий
- Вытекает из сопла при перемещении (travel move)
- Даже с ретрактом капли остаются

**Признаки:**
- Сильный stringing
- Overheating (расплавление мелких деталей)
- Blobs (капли на углах)

### 2. Недостаточный ретракт

**Retraction (ретракт)** — обратное движение филамента, чтобы снизить давление в сопле при перемещении.

**Проблемы:**
- Ретракт слишком маленький (< 4 мм)
- Ретракт слишком медленный (< 30 mm/s)
- Ретракт вообще выключен

### 3. Влажный филамент

**Гигроскопичные материалы** (PLA, PETG, Nylon, TPU) впитывают влагу из воздуха:
- Влага внутри → испаряется при печати
- Пар → пузыри и капли
- Сопло "плюётся" → stringing

**Признаки влажного филамента:**
- Stringing
- Треск/булькание при печати
- Пузыри на поверхности
- Хрупкие детали

### 4. Медленное перемещение (travel speed)

- Чем дольше сопло в воздухе, тем больше вытекает
- Медленный travel → больше stringing

### 5. Длинные перемещения без ретракта

- В слайсерах есть параметр "Minimum travel for retraction"
- Если перемещение короткое → ретракт не срабатывает
- Короткие move без ретракта → волоски

## Решения (по эффективности)

### ⭐⭐⭐ Критичные (попробуйте сначала!)

#### 1. Снизить температуру

**Метод:**
- Понижайте по 5°C за раз
- Печатайте тестовый кубик с мостами и башнями
- Найдите минимальную температуру без under-extrusion

**Таблица оптимальных температур:**

| Материал | Начальная | Оптимальная (против stringing) |
|----------|-----------|-------------------------------|
| PLA | 200–210°C | 190–200°C |
| PLA+ | 210–220°C | 200–210°C |
| PETG | 240–250°C | 230–240°C |
| ABS | 230–240°C | 220–230°C |
| TPU | 220–230°C | 210–220°C |

**⚠️ Внимание:** слишком низкая температура → under-extrusion (зазоры в слоях)

#### 2. Увеличить ретракт (Retraction Distance)

**Рекомендации:**

| Тип экструдера | Ретракт (мм) | Комментарий |
|----------------|--------------|-------------|
| **Direct (прямой)** | 4–6 мм | Короткий путь до сопла |
| **Bowden (боуден)** | 6–10 мм | Длинная PTFE трубка → нужно больше |
| **Гибкие (TPU)** | 1–3 мм | Слишком большой → застревание |

**Метод подбора:**
1. Начните с 5 мм (direct) или 7 мм (bowden)
2. Печатайте тест (2 башни) 
3. Увеличивайте по +0.5 мм, пока stringing не исчезнет
4. **Не превышайте 10 мм!** (риск застревания)

#### 3. Увеличить скорость ретракта (Retraction Speed)

**Рекомендации:**

```
Начальное: 40 mm/s
Оптимальное: 40–60 mm/s (direct), 60–80 mm/s (bowden)
```

**Метод:**
- Увеличивайте по +10 mm/s
- Тестируйте
- Слишком быстро → шестерня экструдера стирает филамент

### ⭐⭐ Очень важные

#### 4. Высушить филамент

**Признаки влаги:**
- Stringing + треск при печати
- Пузыри на поверхности
- Хрупкие детали

**Сушка:**

| Материал | Температура | Время |
|----------|-------------|-------|
| PLA | 50–60°C | 4–6 часов |
| PETG | 65–70°C | 6–8 часов |
| Nylon | 70–80°C | 8–12 часов |
| ABS | 60–70°C | 4–6 часов |
| TPU | 50–60°C | 6–8 часов |

**Где сушить:**
- Дегидратор для продуктов
- Духовка (с термометром!)
- Специальная сушилка для филамента
- НЕ микроволновка!

**Хранение:**
- Герметичный контейнер + силикагель
- Vacuum bags (вакуумные пакеты)
- Сухой бокс (Dry Box) с гигрометром

#### 5. Увеличить Travel Speed (скорость перемещения)

**Настройка:**

```
Обычная скорость печати: 50 mm/s
Travel speed: 150–200 mm/s (в 3–4 раза больше!)
```

**Почему:** чем быстрее сопло перемещается, тем меньше времени вытекать пластику

**⚠️ Предел:** слишком быстро → принтер трясётся, пропускает шаги

#### 6. Z-Hop (подъём сопла при перемещении)

**Что это:** сопло поднимается на 0.2–0.4 мм при travel move

**Плюсы:**
- Не задевает уже напечатанные части
- Меньше "шрамов" на поверхности

**Минусы:**
- Немного медленнее печать
- Может усилить stringing (больше времени в воздухе)

**Рекомендация:**
```
Z-Hop Height: 0.2–0.3 мм
Only when moving over printed parts (PrusaSlicer)
```

#### 7. Coasting (выключение экструзии перед концом линии)

**Что это:** экструдер прекращает подавать филамент за 0.1–0.5 мм до конца линии

**Как работает:**
- Снижает давление в сопле
- Меньше вытекание при travel
- Меньше blobs (капель) в углах

**Настройка (Cura):**
```
Coasting Volume: 0.05–0.2 mm³
Coasting Min Volume: 0.8 mm³
```

**⚠️ Внимание:** слишком много → under-extrusion в конце линий

### ⭐ Дополнительные

#### 8. Wipe (вытирание сопла перед перемещением)

**Что это:** сопло проводит короткое движение по только что напечатанному периметру, вытирая остатки пластика

**Настройка (Cura):**
```
Wipe Distance: 0.5–1.0 мм
```

**Эффект:** капли остаются на внутренней стороне стенки (невидимо)

#### 9. Minimum Travel Distance for Retraction

**Что это:** ретракт срабатывает только если перемещение > X мм

**Проблема:**
- Значение слишком большое → короткие move без ретракта → волоски

**Решение:**
```
Старое: 1.5–2.0 мм
Новое: 0.5–1.0 мм (ретракт чаще)
```

**Компромисс:** больше ретрактов → медленнее печать, износ филамента

#### 10. Combing Mode (маршрутизация перемещений)

**Что это:** слайсер прокладывает travel moves внутри детали (не через пустоту)

**Режимы (PrusaSlicer/Cura):**
- **Off:** прямые перемещения (быстро, но больше stringing)
- **Not in skin:** избегает внешних стенок (баланс)
- **Within infill:** только внутри заполнения (лучше качество, медленнее)

**Рекомендация:** "Not in skin" (лучший баланс)

#### 11. Постобработка (если ничего не помогло)

**Методы удаления stringing:**

1. **Тепловая обработка (Heat gun / фен):**
   - Быстро проведите феном (200–300°C) на расстоянии 5–10 см
   - Волоски расплавляются и исчезают
   - ⚠️ Не перегрейте саму деталь!

2. **Обжиг (Flame treatment):**
   - Быстро провести зажигалкой
   - Эффективно для PLA, PETG
   - ⚠️ Осторожно! Легко испортить

3. **Механическое удаление:**
   - Пальцами (для мягких нитей)
   - Кусачками (для толстых)
   - Наждачкой (для стойких)

4. **Пескоструй (Sandblasting):**
   - Профессиональный метод
   - Удаляет всё, делает матовую поверхность

## Тестирование настроек

**Retraction Test Tower** — специальная модель для проверки ретракта:
- 2 башни с промежутком
- Каждый уровень — разные настройки
- Видно, где stringing исчезает

**Где скачать:**
- Thingiverse: "Retraction Test"
- Teaching Tech: retraction calibration
- Cura/PrusaSlicer: встроенные модели

## Специфика для материалов

### PLA
- Средняя склонность к stringing
- **Решение:** T° 190–200°C, ретракт 5 мм

### PETG
- **Очень склонен к stringing!**
- **Решение:** T° 230–235°C, ретракт 6–7 мм, сухой филамент, Z-hop 0.3 мм

### ABS
- Средняя склонность
- **Решение:** T° 220–230°C, ретракт 5 мм

### TPU (гибкий)
- Высокая склонность
- **Решение:** T° 210–220°C, ретракт 1–3 мм (малый!), медленная печать

### Nylon
- Очень высокая (+ влага!)
- **Решение:** обязательная сушка 8–12 часов, T° 240–250°C, ретракт 6 мм

## Checklist: Устранение Stringing

Проверьте по порядку:

- [ ] **Температура снижена** на 5–10°C от максимальной?
- [ ] **Ретракт включён** и правильно настроен (4–6 мм direct, 6–8 мм bowden)?
- [ ] **Скорость ретракта** 40–60 mm/s?
- [ ] **Филамент сухой?** (нет треска, пузырей)
- [ ] **Travel speed** 150+ mm/s?
- [ ] **Minimum travel for retraction** < 1 мм?
- [ ] **Coasting** включён (0.1 mm³)?
- [ ] **Wipe** включён (0.5 мм)?
