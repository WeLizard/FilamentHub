# Warping (Коробление) - Полное руководство

## Что такое Warping?

**Warping (коробление)** — это дефект 3D-печати, при котором углы детали отлипают от платформы и загибаются вверх. Деталь деформируется, теряет геометрию и может полностью оторваться от стола.

![Warping пример](https://via.placeholder.com/600x300?text=Warping+Example)

## Причины

### 1. Тепловая усадка материала

При остывании пластик сжимается (усадка):
- **PLA:** 0.3–0.5% усадка (минимальная)
- **PETG:** 0.4–0.8% усадка  
- **ABS:** 0.8–1.5% усадка (критично!)
- **Nylon:** 1.0–2.0% усадка

**Что происходит:**
1. Первый слой прилипает к столу (фиксация)
2. Верхние слои остывают и сжимаются
3. Внутреннее напряжение → углы отрываются
4. Деталь загибается вверх (warping)

### 2. Неравномерное охлаждение

- Верхние слои остывают быстрее нижних
- Обдув вентилятором усиливает напряжение
- Большие детали → больше градиент температур

### 3. Плохая адгезия к столу

- Грязная платформа (жир, пыль)
- Низкая температура стола
- Неправильная калибровка (Z-offset)

### 4. Геометрия детали

- Большая площадь дна → больше усадка по краям
- Острые углы → концентрация напряжений
- Тонкие стенки → недостаточная жёсткость

## Как выглядит Warping?

### Лёгкий warping:
- Углы слегка приподняты (1–2 мм)
- Деталь всё ещё на столе
- Геометрия немного нарушена

### Средний warping:
- Углы сильно загнуты (5+ мм)
- Первый слой частично оторван
- Деталь на грани срыва

### Критический warping:
- Деталь полностью оторвалась
- Печать провалена
- "Спагетти" (сопло печатает в воздухе)

## Решения (по эффективности)

### ⭐⭐⭐ Критичные (обязательно!)

#### 1. Подогрев стола (Heated Bed)

**Таблица температур:**

| Материал | Температура стола | Примечание |
|----------|------------------|-----------|
| **PLA** | 50–60°C | Можно без подогрева, но лучше с ним |
| **PETG** | 70–85°C | Обязательно! |
| **ABS** | 95–110°C | Чем выше, тем лучше |
| **ASA** | 95–110°C | Как ABS |
| **Nylon** | 70–90°C | + закрытая камера |
| **PC** | 110–130°C | Требует мощный стол |

**Если нет подогрева:**
- Используйте только PLA
- Brim + клей-карандаш

#### 2. Снизить охлаждение (особенно ABS, ASA)

```
Первые 5 слоёв: вентилятор 0%
Слои 6–10: вентилятор 10–20%
Остальные: вентилятор 30–50% (для ABS), 100% (для PLA)
```

**Для ABS/ASA:** вентилятор полностью ВЫКЛ!

#### 3. Brim (Поля вокруг модели)

**Что это:** дополнительные кольца вокруг детали (как поля шляпы)

**Как работает:**
- Увеличивает площадь контакта со столом
- Держит углы от отлипания
- Легко удаляется после печати

**Настройка в слайсере:**
```
Brim width: 5–10 мм (2–5 линий)
Brim distance: 0 мм (соединён с деталью)
```

**Когда использовать:** всегда для ABS, для больших деталей, для деталей с маленькой площадью контакта

### ⭐⭐ Очень важные

#### 4. Raft (Подложка)

**Что это:** толстая сетка под всей деталью

**Плюсы:**
- Максимальная защита от warping
- Компенсирует неидеальную калибровку

**Минусы:**
- Расход материала
- Некрасивое дно детали
- Сложно удалять

**Когда использовать:** для очень проблемных материалов (ABS, Nylon, PC), если Brim не помогает

#### 5. Закрытая камера (Enclosure)

**Для ABS/ASA/Nylon — критично!**

**Как работает:**
- Поддерживает температуру воздуха 40–50°C
- Равномерное охлаждение
- Нет сквозняков

**DIY решения:**
- Картонная коробка вокруг принтера
- Акриловый корпус
- IKEA LACK стол (популярное решение)

**⚠️ Внимание:** для PLA закрытая камера **ВРЕДНА** (слишком жарко → deformation во время печати)

#### 6. Правильная подготовка платформы

**Для всех материалов:**
1. Очистить спиртом (изопропанол 90%+)
2. Не трогать руками (жир → плохая адгезия)

**Поверхности для разных материалов:**

| Материал | Лучшая поверхность | Дополнительно |
|----------|-------------------|--------------|
| PLA | Стекло, PEI, текстурированный стол | Клей-карандаш (опционально) |
| PETG | PEI, BuildTak, стекло + клей | НЕ голое стекло! |
| ABS | PEI, каптон, стекло + ABS juice | Обязательно подогрев |
| Nylon | PEI, гаролит, стекло + клей | Очень требователен |

**ABS juice (рецепт):**
- Растворить обрезки ABS в ацетоне
- Нанести тонкий слой на стекло
- Идеальная адгезия для ABS

### ⭐ Дополнительные

#### 7. "Мышиные уши" (Mouse Ears)

**Что это:** маленькие диски в углах детали

**Как работает:**
- Увеличивает площадь контакта в углах
- Держит углы от отлипания
- Легко срезать ножом после печати

**Добавление:**
- В слайсере: добавить цилиндры (диаметр 10–15 мм, высота 0.2–0.4 мм) в углы
- Или использовать плагины (Cura: "Auto Towers", PrusaSlicer: встроенная функция)

#### 8. Снизить скорость первого слоя

```
Первый слой: 15–20 mm/s (медленно!)
Остальные: 50–60 mm/s (обычно)
```

**Почему:** медленная печать → лучшая адгезия → меньше шанс отлипания

#### 9. Увеличить температуру первого слоя

```
Первый слой сопла: +5–10°C
Первый слой стола: +5–10°C
```

**Пример для PLA:**
- Первый слой: 210°C (сопло), 65°C (стол)
- Остальные: 200°C (сопло), 60°C (стол)

#### 10. Правильная ориентация детали

**Плохо:**
- Большая площадь дна (вся усадка концентрируется на краях)
- Острые углы снизу

**Хорошо:**
- Маленькая площадь контакта
- Скруглённые углы
- Можно повернуть деталь под углом

#### 11. Отключить обдув для первых слоёв

```
Первые 3–5 слоёв: вентилятор 0%
После: постепенно включать
```

**Особенно для:** PETG, ABS, ASA, Nylon, PC

## Специфика для разных материалов

### PLA
- Warping редко (низкая усадка)
- Обычно достаточно: стол 60°C + очистка спиртом
- Brim для больших деталей

### PETG
- Средний риск warping
- Обязательно: стол 80°C + PEI/клей
- Охлаждение 30–50% (не 100%!)

### ABS/ASA
- **Высокий риск warping!**
- Обязательно: стол 100–110°C + закрытая камера + вентилятор ВЫКЛ
- Brim всегда
- ABS juice на стекло

### Nylon
- **Очень высокий риск warping!**
- Обязательно: стол 80–90°C + закрытая камера + сухой филамент
- Brim/Raft обязательно
- Garolite/PEI платформа

### TPU (гибкий)
- Warping редко (материал гибкий)
- Стол 40–60°C

## Checklist: Устранение Warping

Проверьте по порядку:

- [ ] **Стол подогрет** до нужной температуры?
- [ ] **Платформа чистая** (спирт, без жира)?
- [ ] **Z-offset правильный** (первый слой прижат, но не размазан)?
- [ ] **Brim включён** (5–10 мм)?
- [ ] **Вентилятор выключен** на первых 5 слоях?
- [ ] **Закрытая камера** (для ABS/Nylon)?
- [ ] **Температура стола правильная** для материала?
- [ ] **Деталь правильно ориентирована**?
