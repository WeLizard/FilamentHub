# Захватывание сопла (Nozzle Digging) - Полное руководство

## Что такое Nozzle Digging?

**Захватывание сопла (nozzle digging)** — сопло царапает или цепляется за уже напечатанные слои, оставляя следы или деформируя деталь. Может происходить при ретракте или во время перемещений.

## Признаки проблемы

- Сопло оставляет царапины на поверхности
- Следы на уже напечатанных слоях
- Деформация слоёв при прохождении сопла
- Слои сдвигаются/сминаются
- Пластик "выдавливается" соплом

## Основные причины

### 1. Z-lift при ретракте слишком большой

Слишком высокий Z-lift → сопло поднимается → при возврате может царапать

**Но также:** слишком маленький Z-lift → сопло не поднимается → царапает при перемещении

**Решение:**
- Оптимальный Z-lift: 0.5–1.0 mm
- Не слишком большой (иначе медленнее)
- Не слишком маленький (иначе царапает)

### 2. Неправильная траектория печати (travel path)

Слайсер выбирает путь который проходит через уже напечатанные части

**Решение:**
- Включить "Avoid printed parts when traveling" / "Combing"
- Слайсер будет обходить напечатанные части

### 3. Over-extrusion

Слои слишком толстые → выступают → сопло задевает

**Решение:**
- Уменьшить flow-rate
- См. статью "Over-extrusion"

### 4. Высота слоя слишком большая

Высокие слои → выступают → сопло задевает

**Решение:**
- Уменьшить высоту слоя
- Для сопла 0.4 мм: 0.2 мм оптимально

### 5. Проблемы с Z-осью

Z-ось не поднимается достаточно → сопло слишком близко к слоям

**Решение:**
- Проверить калибровку Z-оси
- Проверить шаги Z-оси
- Проверить что Z-hop работает

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Настроить Z-lift при ретракте**
   - Z-hop: 0.5–1.0 mm (оптимально 0.5–0.7 mm)
   - Не слишком большой, не слишком маленький
   - Включить "Z-hop when retracted"

2. **Включить "Avoid printed parts" / "Combing"**
   - Слайсер будет обходить напечатанные части
   - Предотвращает столкновения

### 🟡 Важно:

3. **Исправить over-extrusion**
   - Уменьшить flow-rate до 95–100%
   - Слои не должны выступать

4. **Уменьшить высоту слоя**
   - 0.2 мм вместо 0.3 мм
   - Более ровные слои

5. **Проверить калибровку Z-оси**
   - Z-ось должна точно подниматься
   - Проверить шаги Z-оси

## Настройки в слайсере

### Cura

```
Z Hop When Retracted: Да
Z Hop Height: 0.5–0.7 mm
Combing Mode: All / Not in Skin
Avoid Printed Parts When Traveling: Да
```

### PrusaSlicer / OrcaSlicer

```
Lift Z: 0.5–0.7 mm
Avoid crossing perimeters: Да
Avoid crossing curled overhangs: Да
```

## Быстрая диагностика

### Сопло царапает при перемещении
→ Z-hop недостаточный или не включён

### Сопло царапает при возврате после ретракта
→ Z-lift слишком большой или траектория неправильная

### Следы на всех слоях
→ Over-extrusion или высота слоя большая

### Следы только в определённых местах
→ Неправильная траектория → включить "Avoid printed parts"

## Checklist

- [ ] Z-hop включён (0.5–0.7 mm)
- [ ] "Avoid printed parts" включено
- [ ] Flow-rate правильный (нет over-extrusion)
- [ ] Высота слоя оптимальная (0.2 mm)
- [ ] Z-ось правильно калибрована
