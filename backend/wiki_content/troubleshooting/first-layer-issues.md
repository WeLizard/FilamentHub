# Проблемы первого слоя (First Layer Issues) - Полное руководство

## Общие проблемы первого слоя

Первый слой критически важен для успешной печати. Если первый слой неправильный — вся печать провалится.

## Типы проблем первого слоя

### 1. Первый слой не прилипает

**Признаки:**
- Первый слой отлипает от стола
- Печать сразу проваливается
- Филамент не прилипает

**Причины и решения:**
- См. статью "Layer Adhesion" (подробно)
- Кратко: очистить стол, увеличить температуру стола, использовать клей/лак, правильно калибровать Z

### 2. Первый слой слишком тонкий (сопло царапает)

**Признаки:**
- Первый слой почти прозрачный
- Сопло царапает стол
- Филамент не выдавливается

**Причины:**
- Сопло слишком близко к столу (Z слишком низкий)
- Z-offset неправильный

**Решение:**
- Поднять Z-калибровку (сопло выше)
- Увеличить Z-offset
- Проверить bed leveling

### 3. Первый слой слишком толстый (не прилипает)

**Признаки:**
- Первый слой не прилипает
- Нити не сплавляются
- Видны отдельные нити

**Причины:**
- Сопло слишком далеко от стола (Z слишком высокий)
- Z-offset неправильный

**Решение:**
- Опустить Z-калибровку (сопло ближе)
- Уменьшить Z-offset
- Проверить bed leveling

### 4. Первый слой неровный (толстый в одном месте, тонкий в другом)

**Признаки:**
- Первый слой то толстый, то тонкий
- Неровная поверхность
- В одном углу сопло царапает, в другом не прилипает

**Причины:**
- Стол не выровнен (bed leveling неправильный)

**Решение:**
- См. статью "Bed Leveling" (подробно)
- Кратко: откалибровать стол (4 угла), использовать mesh leveling

### 5. Первый слой заворачивается (curling)

**Признаки:**
- Углы заворачиваются вверх
- Первый слой не лежит ровно

**Причины:**
- Быстрое охлаждение
- Недостаточная адгезия углов

**Решение:**
- См. статью "Curling Corners" (подробно)
- Кратко: использовать Brim, снизить скорость первого слоя, увеличить температуру стола

### 6. Первый слой "волнами" (elephant's foot)

**Признаки:**
- Нижние слои расширяются
- Слоны ноги на дне детали

**Причины:**
- Сопло слишком близко
- Высокая температура стола
- Первый слой слишком прижат

**Решение:**
- См. статью "Elephant's Foot" (подробно)
- Кратко: поднять Z, снизить температуру стола, включить обдув на втором слое

## Правильный первый слой — как должен выглядеть

### Идеальный первый слой:

- ✅ Нити прижаты к столу, но не расплющены
- ✅ Нити сплавляются друг с другом
- ✅ Ровная поверхность без зазоров
- ✅ Толщина соответствует настройкам (обычно 0.2–0.3 mm)
- ✅ Хорошая адгезия (не отлипает)

### Проверка зазора:

**Используйте лист бумаги:**
- Дом принтера (G28)
- Переместить Z на 0 (G1 Z0)
- Лист бумаги между соплом и столом
- Должен слегка задевать (чувствовать небольшое сопротивление)
- Не должен быть зажат (слишком близко)
- Не должен свободно проходить (слишком далеко)

## Настройки первого слоя в слайсере

### Cura

```
Initial Layer Height: 0.2–0.3 mm
Initial Layer Line Width: 100–120% (немного шире для адгезии)
Initial Layer Speed: 15–20 mm/s (медленно!)
Initial Layer Fan Speed: 0% (выключен!)
Initial Layer Temperature: +5–10°C (немного выше)
```

### PrusaSlicer / OrcaSlicer

```
First layer height: 0.2–0.3 mm
First layer width: 100–120%
First layer speed: 15–20 mm/s
Fan speed (first layer): 0%
First layer temperature: +5–10°C
```

## Пошаговая настройка первого слоя

### Шаг 1: Калибровка стола (Bed Leveling)

1. Дом принтера (G28)
2. Отключить мотора (M84)
3. Калибровать 4 угла (см. "Bed Leveling")
4. Проверить центр

### Шаг 2: Настройка Z-offset

1. Дом принтера (G28)
2. Переместить Z на 0 (G1 Z0)
3. Проверить зазор листом бумаги
4. Настроить Z-offset до правильного зазора
5. Сохранить (M500)

### Шаг 3: Очистка стола

1. Очистить спиртом
2. Убедиться что нет остатков
3. При необходимости использовать клей/лак (для стекла/PEI)

### Шаг 4: Настройки слайсера

1. Первый слой медленнее (15–20 mm/s)
2. Первый слой шире (100–120%)
3. Обдув выключен (0%)
4. Температура стола правильная

### Шаг 5: Тестовая печать

1. Распечатать простой квадрат (20×20 mm, один слой)
2. Проверить что прилипает
3. Проверить что ровный
4. Проверить что нет зазоров
5. Настроить если нужно

## Быстрая диагностика

### Первый слой не прилипает вообще
→ Очистить стол, увеличить температуру стола, использовать клей/лак

### Первый слой прозрачный, сопло царапает
→ Сопло слишком близко → поднять Z

### Первый слой не сплавляется, видны отдельные нити
→ Сопло слишком далеко → опустить Z

### Первый слой неровный (то толстый, то тонкий)
→ Стол не выровнен → калибровка стола

### Первый слой заворачивается в углах
→ Использовать Brim, снизить скорость, увеличить температуру стола

## Checklist

- [ ] Стол откалиброван (4 угла + центр)
- [ ] Z-offset правильно настроен
- [ ] Стол очищен (спирт) и адгезия улучшена (если нужно)
- [ ] Скорость первого слоя снижена (15–20 mm/s)
- [ ] Обдув выключен для первого слоя (0%)
- [ ] Температура стола правильная
- [ ] Тестовая печать первого слоя успешна
