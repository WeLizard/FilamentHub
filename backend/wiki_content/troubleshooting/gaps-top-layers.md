# Пробелы в верхних слоях (Gaps in Top Layers) - Полное руководство

## Что такое пробелы в верхних слоях?

**Пробелы в верхних слоях** — верхняя часть детали пористая, дырявая, с зазорами между нитями. Верхние слои не полностью закрывают деталь, видны промежутки.

## Признаки проблемы

- Пористая верхняя поверхность
- Зазоры/дырки в верхних слоях
- Видны линии заполнения (infill) через верхние слои
- Верхняя часть выглядит как сетка

## Основные причины

### 1. Недостаточно верхних слоёв

Верхних слоёв слишком мало → не закрывают заполнение

**Решение:**
- Увеличить количество верхних слоёв до минимум 6
- Рекомендуется: 6–10 слоёв для надёжности

**Настройка в слайсере:**
```
Top Layers: 6–10 (вместо 3–4)
```

### 2. Низкий процент заполнения (Infill)

Мало заполнения → верхние слои провисают между опорами

**Решение:**
- Увеличить заполнение до минимум 15–20%
- Для надёжных деталей: 30–50%
- Для декоративных: 10–15% (но больше верхних слоёв)

**Настройка в слайсере:**
```
Infill: 20–30% (минимум 15%)
```

### 3. Недостаточный обдув

Верхние слои плохо остывают → провисают

**Решение:**
- Включить обдув для верхних слоёв (50–100%)
- Особенно важно для PLA

### 4. Низкая температура

Пластик недостаточно жидкий → плохо заполняет промежутки

**Решение:**
- Увеличить температуру сопла на 5°C
- Проверить оптимальную температуру материала

### 5. Недостаточная экструзия (under-extrusion)

Недостаточно пластика → зазоры в верхних слоях

**Решение:**
- Увеличить flow-rate до 100–105%
- Проверить сопло (не забито)
- См. статью "Under-extrusion"

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Увеличить количество верхних слоёв**
   - Минимум 6 слоёв (рекомендуется 8–10)
   - Больше слоёв = надёжнее закрытие

2. **Увеличить заполнение (Infill)**
   - Минимум 15–20%
   - Больше заполнения = меньше провисание верхних слоёв

### 🟡 Важно:

3. **Включить обдув для верхних слоёв**
   - 50–100% для PLA
   - 30–50% для PETG
   - Важно для быстрого остывания верхних слоёв

4. **Увеличить температуру сопла**
   - На 5°C
   - Лучшее заполнение промежутков

5. **Проверить flow-rate**
   - Увеличить до 100–105% если нужно
   - Убедиться что нет under-extrusion

### 🟢 Дополнительно:

6. **Использовать "Ironing" (утюжка) в слайсере**
   - Проходит соплом по верхней поверхности без экструзии
   - Разглаживает верхние слои
   - Доступно в Cura

7. **Увеличить ширину линии верхних слоёв**
   - В некоторых слайсерах можно настроить
   - Шире линии = лучше покрытие

## Настройки в слайсере

### Cura

```
Top Layers: 8
Bottom Layers: 5
Infill: 20%
Top/Bottom Line Width: 0.4 mm (или немного больше)
Ironing: Включено (опционально)
```

### PrusaSlicer / OrcaSlicer

```
Top solid layers: 8
Bottom solid layers: 5
Fill density: 20%
```

## Быстрая диагностика

### Пористая верхняя поверхность, видны дыры
→ Недостаточно верхних слоёв или низкий infill

### Видны линии заполнения через верх
→ Мало верхних слоёв

### Верхняя часть провисшая
→ Низкий infill или нет обдува

### Зазоры между нитями верхних слоёв
→ Under-extrusion или низкая температура

## Специфика для разных материалов

### PLA
```
Top Layers: 8
Infill: 20%
Обдув: 100% (для верхних слоёв)
```

### PETG
```
Top Layers: 8
Infill: 25%
Обдув: 40–50% (НЕ 100%!)
```

### ABS
```
Top Layers: 10
Infill: 30%
Обдув: 0% (выключен)
```

## Checklist

- [ ] Верхних слоёв минимум 6 (лучше 8–10)
- [ ] Заполнение минимум 15–20%
- [ ] Обдув включён для верхних слоёв
- [ ] Температура сопла правильная (+5°C если нужно)
- [ ] Flow-rate 100% (проверить under-extrusion)
