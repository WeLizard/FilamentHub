# Плохие мосты (Poor Bridging) - Полное руководство

## Что такое Bridging?

**Bridging (мосты)** — печать горизонтальных участков между двумя опорными точками без поддержек снизу. Мосты провисают или рвутся если не настроены правильно.

## Признаки проблемы

- Мосты провисают (sagging)
- Нити мостов рвутся
- Неровная нижняя поверхность мостов
- Дырки в мостах

## Основные причины

### 1. Недостаточное охлаждение

Мосты нужно остужать быстро иначе провисают

**Решение:**
- Максимальный обдув для мостов (100% для PLA)
- Скорость обдува максимальная
- Охлаждение начинается сразу

### 2. Скорость моста слишком медленная

Слишком медленная печать → больше времени провисания

**Решение:**
- Увеличить скорость печати мостов
- Обычно 30–50 mm/s для мостов
- Быстрее = лучше (в разумных пределах)

### 3. Температура слишком высокая

Пластик слишком жидкий → провисает

**Решение:**
- Снизить температуру для мостов на 5–10°C
- В некоторых слайсерах есть "Bridge Temperature"

### 4. Длина моста слишком большая

Длинные мосты (> 20–30 мм) всегда провисают

**Решение:**
- Использовать поддержки для длинных мостов
- Или переориентировать модель
- Или разделить модель

### 5. Недостаточная экструзия

Недостаточно пластика → мост рвётся

**Решение:**
- Увеличить flow-rate для мостов
- Проверить что нет under-extrusion

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Максимальный обдув для мостов**
   - 100% для PLA
   - 50–70% для PETG
   - Быстрое охлаждение = меньше провисание

2. **Увеличить скорость печати мостов**
   - 40–60 mm/s (быстрее чем обычная печать)
   - Быстрая печать = меньше времени на провисание

3. **Снизить температуру для мостов**
   - На 5–10°C от обычной температуры
   - Более вязкий пластик = меньше провисание

### 🟡 Важно:

4. **Использовать поддержки для длинных мостов**
   - Если мост > 20–30 мм
   - Или переориентировать модель

5. **Увеличить flow-rate для мостов (опционально)**
   - 105–110% для лучшего заполнения
   - Но осторожно (может усилить провисание)

6. **Уменьшить высоту слоя**
   - 0.15–0.2 мм для лучших мостов
   - Более тонкие слои = прочнее мосты

## Настройки в слайсере

### Cura

```
Bridge Settings:
- Bridge Wall Speed: 40 mm/s
- Bridge Wall Flow: 100%
- Bridge Skin Speed: 40 mm/s
- Bridge Skin Flow: 100%
- Bridge Fan Speed: 100%
- Enable Bridge Settings: Да
```

### PrusaSlicer / OrcaSlicer

```
Bridge Settings:
- Bridge speed: 40 mm/s
- Bridge flow ratio: 1.0 (100%)
- Bridge fan speed: 100%
```

## Ограничения мостов

### Максимальная длина без провисания:

| Материал | Максимальная длина моста |
|----------|-------------------------|
| PLA | 20–30 мм (с правильными настройками) |
| PETG | 15–25 мм |
| ABS | 10–20 мм (без обдува) |

**Для более длинных мостов:** использовать поддержки

## Быстрая диагностика

### Мосты провисают равномерно
→ Недостаточно охлаждения или высокая температура

### Мосты рвутся
→ Недостаточная экструзия или слишком быстро

### Мосты провисают только длинные
→ Физическое ограничение → использовать поддержки

## Специфика для разных материалов

### PLA
```
Обдув: 100%
Скорость моста: 40–50 mm/s
Температура: -5–10°C для мостов
Лучший материал для мостов!
```

### PETG
```
Обдув: 50–70% (не 100%!)
Скорость моста: 40 mm/s
Температура: -5°C для мостов
Сложнее чем PLA
```

### ABS
```
Обдув: 0% (выключен!)
Скорость моста: 30 mm/s
Температура: стандартная
Сложнее всего (без обдува)
```

## Checklist

- [ ] Максимальный обдув для мостов (100% для PLA)
- [ ] Скорость мостов увеличена (40–50 mm/s)
- [ ] Температура снижена для мостов (-5–10°C)
- [ ] Длинные мосты (> 30 мм) имеют поддержки
- [ ] Flow-rate правильный (100% или немного выше)
