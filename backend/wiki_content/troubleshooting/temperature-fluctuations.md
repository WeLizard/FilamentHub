# Колебания температуры (Temperature Fluctuations) - Полное руководство

## Что такое Temperature Fluctuations?

**Колебания температуры (temperature fluctuations)** — температура сопла или стола нестабильна, колеблется вверх-вниз. Это приводит к непостоянной экструзии и проблемам с качеством.

## Признаки проблемы

- Температура сопла колеблется (например, 200°C ± 5°C)
- Непостоянная экструзия (то больше, то меньше)
- Проблемы с качеством (непостоянное качество)
- Принтер не может удерживать заданную температуру

## Основные причины

### 1. Проблемы с термистором

Термистор (датчик температуры) работает неправильно

**Причины:**
- Термистор неправильно установлен (не касается нагревателя)
- Термистор повреждён
- Плохой контакт проводов термистора

**Решение:**
- Проверить установку термистора (должен касаться нагревателя)
- Проверить провода (нет обрывов, хороший контакт)
- Заменить термистор (если повреждён)

### 2. Проблемы с нагревателем

Нагреватель работает нестабильно

**Причины:**
- Нагреватель изношён
- Плохой контакт проводов нагревателя
- Недостаточная мощность нагревателя

**Решение:**
- Проверить провода нагревателя (хороший контакт)
- Заменить нагреватель (если изношён)
- Проверить мощность нагревателя

### 3. Сквозняки

Сквозняки охлаждают HotEnd → температура падает

**Решение:**
- Закрыть принтер (корпус)
- Убрать сквозняки (вентиляторы, открытые окна)
- Использовать защиту HotEnd

### 4. Недостаточная изоляция HotEnd

HotEnd теряет тепло → температура нестабильна

**Решение:**
- Использовать изоляцию HotEnd (термостойкая лента, войлок)
- Проверить что изоляция не повреждена

### 5. PID настройки неправильные

PID (регулятор температуры) настроен неправильно → нестабильная температура

**Решение:**
- Перекалибровать PID
- Использовать команду PID autotune

## Решения по приоритету

### 🔴 Критично:

1. **Проверить термистор**
   - Правильно установлен (касается нагревателя)
   - Провода не повреждены
   - Хороший контакт

2. **Проверить нагреватель**
   - Провода не повреждены
   - Хороший контакт
   - Заменить если изношён

3. **Перекалибровать PID**
   - Запустить PID autotune
   - Сохранить настройки

### 🟡 Важно:

4. **Убрать сквозняки**
   - Закрыть принтер
   - Убрать вентиляторы рядом с принтером

5. **Использовать изоляцию HotEnd**
   - Термостойкая лента
   - Войлок (если разрешено)
   - Проверить что изоляция целая

6. **Проверить мощность нагревателя**
   - Достаточная мощность для материала
   - Не перегружен

## Калибровка PID

### Что такое PID:

PID (Proportional-Integral-Derivative) — алгоритм контроля температуры. Правильные настройки = стабильная температура.

### Как перекалибровать (Marlin):

1. Нагреть HotEnd до рабочей температуры (например, 200°C для PLA)
2. Запустить PID autotune:
   ```
   M303 E0 S200 C8
   ```
   - E0 = экструдер 0
   - S200 = температура 200°C
   - C8 = 8 циклов
3. Дождаться завершения (несколько минут)
4. Сохранить настройки:
   ```
   M500
   ```
5. Перезагрузить принтер

### Для стола:

```
M303 E-1 S60 C8
```
- E-1 = стол (bed)

## Быстрая диагностика

### Температура колеблется ±2–3°C
→ Нормально (допустимые колебания)

### Температура колеблется ±5°C и больше
→ Проблема → проверить термистор, нагреватель, PID

### Температура не может достичь заданной
→ Мощность нагревателя недостаточна или сквозняки

### Температура постоянно растёт (thermal runaway)
→ Неисправный термистор или нагреватель → ОСТАНОВИТЬ ПРИНТЕР!

## Допустимые колебания

### Нормальные колебания:

- ±2–3°C: нормально
- ±5°C: приемлемо, но можно улучшить
- ±10°C и больше: проблема!

### Проверка:

- Наблюдать за температурой в течение 5–10 минут
- Температура должна быть стабильной (±2–3°C)

## Checklist

- [ ] Термистор правильно установлен (касается нагревателя)
- [ ] Провода термистора не повреждены (хороший контакт)
- [ ] Нагреватель работает (провода не повреждены)
- [ ] PID перекалиброван (autotune)
- [ ] Нет сквозняков (принтер закрыт)
- [ ] HotEnd изолирован (если нужно)

## ⚠️ Безопасность

### Thermal Runaway Protection:

Всегда должен быть включён! Защищает от перегрева если термистор или нагреватель неисправны.

**Проверить:**
- Включена ли защита в прошивке
- Работает ли (тест: отключить термистор → принтер должен остановиться)
