# Проблемы с заполнением (Infill Issues) - Полное руководство

## Типы проблем с заполнением

### 1. Слабое заполнение (Weak Infill)

**Признаки:**
- Заполнение легко ломается
- Деталь хрупкая
- Заполнение тонкое, не держит форму

**Причины:**
- Недостаточная экструзия
- Низкая температура
- Слишком быстрая скорость заполнения

**Решение:**
- Увеличить flow-rate до 100%
- Поднять температуру на 5°C
- Снизить скорость заполнения
- Увеличить процент заполнения (15–30%)

### 2. Деформированное заполнение (Deformed Infill)

**Признаки:**
- Заполнение неровное, деформированное
- Линии заполнения не соединяются
- Странные формы заполнения

**Причины:**
- Слишком быстрая скорость
- Недостаточная экструзия
- Высокая температура

**Решение:**
- Снизить скорость заполнения
- Увеличить flow-rate
- Снизить температуру на 5°C

### 3. Заполнение видно через стенки (Infill Poking Through)

**Признаки:**
- Линии заполнения видны на внешних стенках
- Бугры на стенках от заполнения
- Неровная поверхность

**Причины:**
- Слишком высокий процент заполнения
- Тонкие стенки
- Заполнение слишком близко к стенкам

**Решение:**
- Уменьшить процент заполнения
- Увеличить количество слоёв стенок (3–4 вместо 2)
- Использовать "Infill Overlap" на минимум (5–10%)
- Использовать "Infill before walls" = НЕТ

## Решения по приоритету

### 🔴 Критично:

1. **Проверить flow-rate и температуру**
   - Flow-rate: 100%
   - Температура оптимальная (+5°C если нужно)

2. **Настроить скорость заполнения**
   - Заполнение можно печатать быстрее чем стены
   - Но не слишком быстро (иначе деформируется)
   - Обычно: 60–80 mm/s для заполнения

3. **Увеличить количество стенок**
   - 3–4 слоя стенок
   - Заполнение не будет видно снаружи

### 🟡 Важно:

4. **Настроить "Infill Overlap"**
   - Уменьшить до 5–10% (по умолчанию может быть 20–30%)
   - Меньше перекрытие = заполнение не давит на стенки

5. **Использовать правильный паттерн заполнения**
   - Grid, Lines — быстро
   - Triangles, Cubic — прочно
   - Gyroid — хорошо для гибкости

## Настройки в слайсере

### Cura

```
Infill Density: 15–30%
Infill Pattern: Grid / Cubic
Infill Speed: 60–80 mm/s
Infill Overlap: 5–10%
Infill Before Walls: Нет
Wall Line Count: 3–4
```

### PrusaSlicer / OrcaSlicer

```
Fill density: 15–30%
Fill pattern: Grid / Stars
Infill speed: 60–80 mm/s
Infill/perimeters overlap: 5–10%
External perimeters first: Да
Perimeters: 3–4
```

## Рекомендуемые проценты заполнения

| Применение | Процент заполнения |
|-----------|-------------------|
| Декоративные детали | 5–15% |
| Обычные детали | 15–25% |
| Функциональные детали | 30–50% |
| Прочные детали | 50–80% |
| Максимальная прочность | 80–100% |

## Быстрая диагностика

### Заполнение слабое, ломается
→ Недостаточная экструзия или низкая температура

### Заполнение деформированное
→ Слишком быстрая скорость или высокая температура

### Заполнение видно через стенки
→ Тонкие стенки или слишком высокий overlap

## Checklist

- [ ] Flow-rate 100%
- [ ] Температура оптимальная
- [ ] Скорость заполнения разумная (60–80 mm/s)
- [ ] Стенок минимум 3–4 слоя
- [ ] Infill Overlap уменьшен (5–10%)
- [ ] "Infill Before Walls" = Нет
