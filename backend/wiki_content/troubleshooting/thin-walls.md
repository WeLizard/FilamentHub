# Тонкие стенки не печатаются - Полное руководство

## Что такое проблема тонких стенок?

**Тонкие стенки не печатаются** — детали тоньше диаметра сопла печатаются прерывисто или вообще не печатаются. Сопло физически не может напечатать линию тоньше своего диаметра.

## Признаки проблемы

- Тонкие элементы (менее 0.4 мм для сопла 0.4 мм) прерывистые
- Щели в тонких стенках
- Детали тоньше сопла не печатаются вообще
- Мелкие детали получаются неправильной формы

## Основные причины

### 1. Толщина стенки меньше диаметра сопла

**Физическое ограничение FDM печати!**

Сопло 0.4 мм не может напечатать линию тоньше 0.4 мм

**Минимальная толщина:**
- Сопло 0.4 мм → минимум 0.4 мм (лучше 0.5–0.6 мм)
- Сопло 0.3 мм → минимум 0.3 мм (лучше 0.4 мм)
- Сопло 0.2 мм → минимум 0.2 мм (лучше 0.3 мм)

**Решение:**
- Утолстить модель (минимум 1.2 мм для надёжности)
- Использовать меньшее сопло (0.2–0.3 мм)
- Переделать модель с учётом ограничений

### 2. Неправильная настройка "Detect Thin Walls" в слайсере

Слайсер игнорирует тонкие элементы

**Решение:**
- Включить "Detect Thin Walls" / "Print Thin Walls" в слайсере
- Cura: Settings → Print Thin Walls
- PrusaSlicer: Print Settings → Layers and perimeters → Detect thin walls

### 3. Толщина стенки не кратна диаметру сопла

Например, стенка 0.5 мм при сопле 0.4 мм → одна линия 0.4 мм + зазор 0.1 мм

**Решение:**
- Толщина стенки должна быть кратна диаметру сопла
- Для сопла 0.4 мм: 0.4, 0.8, 1.2, 1.6 мм
- Лучше: 1.2–1.6 мм для надёжности

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Утолстить модель**
   - Минимум 1.2 мм для сопла 0.4 мм
   - Использовать "Shell" в CAD программе
   - Или "Offset" в слайсере для увеличения толщины

2. **Включить "Print Thin Walls" в слайсере**
   - Cura: Print Thin Walls
   - PrusaSlicer: Detect thin walls
   - Может помочь, но не решает физическое ограничение

### 🟡 Важно:

3. **Использовать меньшее сопло**
   - Сопло 0.3 мм → минимум 0.3 мм
   - Сопло 0.2 мм → минимум 0.2 мм
   - Минус: медленнее печать

4. **Проверить настройки стенок в слайсере**
   - Minimum wall line width
   - Должно соответствовать диаметру сопла

5. **Переделать модель**
   - Учесть ограничения FDM печати
   - Минимальная толщина = диаметр сопла + запас

## Настройки в слайсере

### Cura

```
Print Thin Walls: Включено
Minimum Wall Line Width: 0.4 mm (для сопла 0.4 мм)
Wall Line Count: Рассчитывается автоматически
```

### PrusaSlicer / OrcaSlicer

```
Detect thin walls: Да
Minimum perimeters: Автоматически
External perimeters width: 0.4 mm
```

## Быстрая диагностика

### Тонкие элементы вообще не печатаются
→ Толщина меньше диаметра сопла → утолстить модель

### Тонкие элементы прерывистые
→ Толщина близка к диаметру сопла → утолстить или использовать меньшее сопло

### Мелкие детали неправильной формы
→ Физическое ограничение → переделать модель

## Рекомендуемые минимальные толщины

| Диаметр сопла | Минимальная толщина | Рекомендуемая толщина |
|--------------|---------------------|----------------------|
| 0.2 мм | 0.2 мм | 0.4–0.6 мм |
| 0.3 мм | 0.3 мм | 0.5–0.8 мм |
| 0.4 мм | 0.4 мм | 0.8–1.2 мм |
| 0.5 мм | 0.5 мм | 1.0–1.5 мм |
| 0.6 мм | 0.6 мм | 1.2–1.8 мм |

## Как утолстить модель

### В CAD программе:
- Использовать "Shell" / "Thicken"
- Увеличить толщину стенок
- Проверить что все элементы достаточно толстые

### В слайсере:
- Некоторые слайсеры имеют "Offset" для стенок
- Но лучше исправить модель в CAD

## Checklist

- [ ] Толщина стенок минимум 1.2 мм (для сопла 0.4 мм)
- [ ] "Print Thin Walls" включено в слайсере
- [ ] Диаметр сопла соответствует требованиям модели
- [ ] Модель проверена на тонкие элементы перед печатью
