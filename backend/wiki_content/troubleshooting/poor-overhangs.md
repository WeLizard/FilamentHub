# Плохие нависы (Poor Overhangs) - Полное руководство

## Что такое Overhangs?

**Нависы (overhangs)** — части детали которые печатаются без поддержки снизу, нависают над пустотой. Плохие нависы провисают, нити рвутся, поверхность неровная.

## Признаки проблемы

- Нависы провисают (sagging)
- Нити нависов рвутся
- Неровная нижняя поверхность нависов
- "Ступенчатость" на нависах
- Нити свисают вниз

## Основные причины

### 1. Угол нависа слишком большой

**Физическое ограничение FDM!**

- **Без поддержек:** максимум 45–60° (зависит от материала)
- **С поддержками:** любой угол возможен

**Ограничения по материалам:**
- PLA: 45–50° (лучший для нависов)
- PETG: 40–45°
- ABS: 35–40° (худший)

### 2. Недостаточное охлаждение

Нависы нужно остужать быстро, иначе провисают

**Решение:**
- Максимальный обдув (100% для PLA)
- Особенно важно для нависов

### 3. Высота слоя слишком большая

Толстые слои → хуже качество нависов

**Решение:**
- Уменьшить высоту слоя до 0.15–0.2 mm
- Более тонкие слои = лучшие нависы

### 4. Температура слишком высокая

Пластик слишком жидкий → провисает

**Решение:**
- Снизить температуру на 5–10°C
- Более вязкий пластик = меньше провисание

### 5. Скорость печати слишком быстрая

Быстрая печать → хуже качество нависов

**Решение:**
- Снизить скорость для нависов
- Особенно внешние стены с нависами

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Использовать поддержки для больших углов**
   - Углы > 45° (для PLA) → использовать поддержки
   - Настройка в слайсере: "Support Overhang Angle"

2. **Максимальный обдув**
   - 100% для PLA
   - 50–70% для PETG
   - Критично для нависов!

3. **Снизить температуру на 5–10°C**
   - Более вязкий пластик
   - Меньше провисание

### 🟡 Важно:

4. **Уменьшить высоту слоя**
   - 0.15–0.2 mm для лучших нависов
   - Более тонкие слои = лучше качество

5. **Снизить скорость для нависов**
   - 30–40 mm/s для внешних стенок с нависами
   - Медленнее = лучше качество

6. **Использовать "Overhang Speed" в слайсере**
   - Отдельная скорость для нависов (если доступно)
   - Обычно 50% от обычной скорости

## Настройки в слайсере

### Cura

```
Support Overhang Angle: 45–50° (для PLA)
Overhang Speed: 20–30 mm/s (50% от обычной)
Fan Speed: 100% (для PLA)
Layer Height: 0.15–0.2 mm
```

### PrusaSlicer / OrcaSlicer

```
Support threshold angle: 45° (для PLA)
External perimeters speed: 30–40 mm/s
Fan speed: 100% (для PLA)
Layer height: 0.15–0.2 mm
```

## Ограничения нависов без поддержек

### Максимальные углы:

| Материал | Максимальный угол без поддержек |
|----------|--------------------------------|
| PLA | 45–50° |
| PETG | 40–45° |
| ABS | 35–40° |
| Nylon | 40–45° |

**Для больших углов:** обязательно использовать поддержки!

## Быстрая диагностика

### Нависы провисают равномерно
→ Недостаточно охлаждения или высокая температура

### Нависы рвутся
→ Слишком большой угол или недостаточная экструзия

### Нависы провисают только большие углы (> 45°)
→ Физическое ограничение → использовать поддержки

## Специфика для разных материалов

### PLA
```
Максимальный угол: 45–50°
Обдув: 100%
Температура: стандартная или -5°C
Лучший материал для нависов!
```

### PETG
```
Максимальный угол: 40–45°
Обдув: 50–70% (не 100%!)
Температура: стандартная или -5°C
Сложнее чем PLA
```

### ABS
```
Максимальный угол: 35–40°
Обдув: 0% (выключен!)
Температура: стандартная
Самый сложный для нависов (без обдува)
```

## Поддержки для нависов

### Когда использовать:

- Углы > 45° (для PLA)
- Углы > 40° (для PETG/ABS)
- Сложные нависы

### Настройки поддержек:

```
Support Overhang Angle: 45–50°
Support Pattern: Grid / Lines
Support Density: 15–20% (достаточно)
Support Interface: Да (плотный интерфейс)
```

## Checklist

- [ ] Угол нависа ≤ 45° (или использовать поддержки)
- [ ] Максимальный обдув (100% для PLA)
- [ ] Температура снижена (-5–10°C если нужно)
- [ ] Высота слоя уменьшена (0.15–0.2 mm)
- [ ] Скорость снижена для нависов (30–40 mm/s)
- [ ] Поддержки включены для больших углов
