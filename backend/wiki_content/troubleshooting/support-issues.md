# Проблемы с поддержками (Support Issues) - Полное руководство

## Типы проблем с поддержками

### 1. Поддержки слишком сложно удалять

**Признаки:**
- Поддержки "впаиваются" в модель
- Невозможно удалить без повреждения детали
- Остатки поддержек на поверхности

**Причины:**
- Поддержки слишком близко к модели
- Высокая плотность поддержек
- Одинаковый материал для модели и поддержек

**Решение:**
- Увеличить "Support Z Distance" (зазор между поддержками и моделью)
- 0.2–0.3 mm для слоя 0.2 mm
- Уменьшить плотность поддержек (15–20% достаточно)
- Использовать растворимые поддержки (PVA для PLA, HIPS для ABS)

### 2. Плохое качество поверхности над поддержками

**Признаки:**
- Шероховатая поверхность где были поддержки
- Видны отпечатки поддержек
- Неровная нижняя поверхность

**Причины:**
- Поддержки слишком далеко (провисают)
- Недостаточная плотность интерфейса поддержек

**Решение:**
- Уменьшить "Support Z Distance" (но не слишком!)
- Использовать "Support Interface" (плотный интерфейс)
- Interface layers: 3–4 слоя
- Interface density: 50–70%

### 3. Поддержки не поддерживают

**Признаки:**
- Поддержки падают/ломаются
- Модель провисает
- Печать проваливается

**Причины:**
- Слишком низкая плотность поддержек
- Слишком хрупкий материал поддержек
- Неправильная геометрия поддержек

**Решение:**
- Увеличить плотность поддержек до 20–30%
- Использовать более прочный паттерн (Grid вместо Lines)
- Проверить что поддержки правильно генерируются

## Решения по приоритету

### 🔴 Критично:

1. **Настроить "Support Z Distance"**
   - Зазор между поддержками и моделью
   - 0.2 mm для слоя 0.2 mm (1 слой)
   - 0.3 mm для слоя 0.3 mm (1 слой)
   - Не слишком близко (трудно удалить)
   - Не слишком далеко (провисает)

2. **Использовать Support Interface**
   - Плотный интерфейс между поддержками и моделью
   - Interface layers: 3–4
   - Interface density: 50–70%
   - Лучшее качество поверхности

3. **Оптимизировать плотность поддержек**
   - 15–20% обычно достаточно
   - Больше = сложнее удалить
   - Меньше = может не поддерживать

### 🟡 Важно:

4. **Использовать правильный паттерн**
   - Grid, Triangles — прочнее
   - Lines, Zig-zag — легче удалить
   - Компромисс: Grid для прочности

5. **Настроить "Support Overhang Angle"**
   - Угол при котором генерируются поддержки
   - 45–50° для PLA
   - Меньше = больше поддержек (лучше качество, сложнее удалить)

6. **Использовать Tree Supports (если доступно)**
   - Органические формы поддержек
   - Легче удалять
   - Меньше материала

## Настройки в слайсере

### Cura

```
Support: Включено
Support Overhang Angle: 45–50°
Support Pattern: Grid
Support Density: 15–20%
Support Z Distance: 0.2 mm
Support Interface: Да
Support Interface Layers: 3–4
Support Interface Density: 50–70%
Support Roof: Да (опционально)
Support Floor: Да (опционально)
```

### PrusaSlicer / OrcaSlicer

```
Generate supports: Да
Support on build plate only: Нет (если нужны поддержки внутри)
Support threshold angle: 45°
Support pattern: Grid
Support density: 15–20%
Contact Z distance: 0.2 mm
Interface layers: 3–4
Interface pattern spacing: 0.5 mm
```

## Быстрая диагностика

### Поддержки слишком сложно удалить
→ Увеличить Support Z Distance (0.2 → 0.3 mm)

### Поверхность плохая над поддержками
→ Использовать Support Interface (плотный интерфейс)

### Поддержки не держат модель
→ Увеличить плотность (15% → 25%) или использовать более прочный паттерн

## Специфика для разных материалов

### PLA

```
Support Z Distance: 0.2 mm
Support Material: PLA (такой же) или PVA (растворимый)
Легко удалять (хрупкий материал)
```

### PETG

```
Support Z Distance: 0.2–0.3 mm
Support Material: PETG (такой же)
Средняя сложность удаления
```

### ABS

```
Support Z Distance: 0.2 mm
Support Material: ABS (такой же) или HIPS (растворимый в лимонене)
Можно растворить HIPS поддержки
```

## Растворимые поддержки

### PVA (для PLA)

- Растворяется в воде
- Идеально для сложных поддержек
- Дороже
- Требует принтер с двумя экструдерами

### HIPS (для ABS)

- Растворяется в лимонене (d-лимонен)
- Идеально для ABS
- Требует принтер с двумя экструдерами

## Checklist

- [ ] Support Z Distance правильный (0.2–0.3 mm)
- [ ] Support Interface включён (3–4 слоя, 50–70% плотность)
- [ ] Плотность поддержек оптимальная (15–20%)
- [ ] Паттерн поддержек правильный (Grid для прочности)
- [ ] Support Overhang Angle настроен (45–50°)
