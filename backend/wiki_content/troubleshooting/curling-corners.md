# Заворачивание углов (Curling Corners) - Полное руководство

## Что такое Curling Corners?

**Заворачивание углов (curling corners)** — углы детали заворачиваются вверх, особенно на первом слое. Похоже на warping, но происходит локально на углах.

## Признаки проблемы

- Углы заворачиваются вверх
- Особенно на первом слое
- Деталь не лежит ровно
- Углы отлипают от стола

## Основные причины

### 1. Недостаточная адгезия углов

Углы остывают быстрее → сжимаются → заворачиваются

**Решение:**
- Увеличить температуру стола
- Использовать Brim или Raft
- Улучшить адгезию (клей, лак для волос, PEI)

### 2. Слишком быстрое охлаждение

Резкое охлаждение → напряжение → заворачивание

**Решение:**
- Закрыть камеру принтера (для ABS)
- Снизить обдув для первого слоя
- Использовать закрытый принтер

### 3. Температура стола слишком высокая

Первый слой слишком горячий → при остывании заворачивается

**Решение:**
- Снизить температуру стола на 5–10°C
- Или снизить температуру после первого слоя

### 4. Скорость первого слоя слишком высокая

Быстрая печать → первый слой не успевает прилипнуть

**Решение:**
- Снизить скорость первого слоя до 15–20 mm/s
- Дать слою время прилипнуть

### 5. Недостаточная площадь контакта

Маленькие углы → мало площади контакта → легко отлипают

**Решение:**
- Использовать Brim (юбка вокруг модели)
- Увеличить площадь контакта углов

## Решения по приоритету

### 🔴 Критично:

1. **Использовать Brim (юбка)**
   - Увеличивает площадь контакта углов
   - Brim width: 5–10 mm
   - Легко удаляется после печати

2. **Снизить скорость первого слоя**
   - 15–20 mm/s для первого слоя
   - Дать время прилипнуть

3. **Увеличить температуру стола**
   - На 5–10°C от обычной
   - Лучшая адгезия

### 🟡 Важно:

4. **Улучшить адгезию стола**
   - Клей для стола (для стекла)
   - Лак для волос (для PEI)
   - Очистить стол (спирт)

5. **Снизить обдув для первого слоя**
   - 0% для первого слоя
   - Потом включить

6. **Использовать Raft (плот)**
   - Если Brim не помогает
   - Больше материала, сложнее удалить

```
First Layer Speed: 15–20 mm/s
Brim Width: 5–10 mm
Brim Type: Outer only
Initial Layer Fan Speed: 0%
Bed Temperature: +5–10°C (от обычной)
```

### PrusaSlicer / OrcaSlicer

```
First layer speed: 15–20 mm/s
Brim width: 5–10 mm
Fan speed (first layer): 0%
Bed temperature: +5–10°C
```

## Быстрая диагностика

### Углы заворачиваются на первом слое
→ Недостаточная адгезия → Brim + снизить скорость первого слоя

### Углы заворачиваются только на больших деталях
→ Быстрое охлаждение → закрыть камеру (для ABS)

### Углы заворачиваются на всех материалах
→ Проблема с адгезией стола → очистить + клей/лак

## Специфика для разных материалов

### PLA

```
Bed Temperature: 60–70°C
Brim: 5–10 mm
First Layer Speed: 15–20 mm/s
Fan: 0% для первого слоя, потом 100%
```

### PETG

```
Bed Temperature: 70–80°C
Brim: 5–10 mm
First Layer Speed: 15 mm/s
Fan: 0% для первого слоя, потом 50%
```

### ABS

```
Bed Temperature: 90–100°C
Brim: 10 mm (больше!)
First Layer Speed: 15 mm/s
Fan: 0% (выключен!)
Закрытая камера: 50–60°C
```

## Brim vs Raft

### Brim (юбка):
- Тонкая рамка вокруг модели
- Легко удаляется
- Увеличивает площадь контакта углов
- Рекомендуется для curling

### Raft (плот):
- Толстая подложка под всей моделью
- Сложнее удалить
- Больше материала
- Использовать если Brim не помогает

## Checklist

- [ ] Brim включён (5–10 mm ширины)
- [ ] Скорость первого слоя снижена (15–20 mm/s)
- [ ] Температура стола увеличена (+5–10°C)
- [ ] Обдув выключен для первого слоя (0%)
- [ ] Стол очищен и адгезия улучшена (клей/лак)
- [ ] Закрытая камера для ABS
