# Плохая адгезия первого слоя - Полное руководство

## Что такое плохая адгезия?

**Плохая адгезия первого слоя** — первый слой не прилипает к платформе, печать сразу проваливается или отлипает во время печати. Критическая проблема, без решения которой печать невозможна.

## Признаки проблемы

- Первый слой не прилипает к столу
- Нити пластика не держатся, сдуваются
- Печать сразу проваливается
- Углы отлипают во время печати
- Зазоры между нитями первого слоя

## Основные причины

### 1. Неправильная калибровка Z-offset

**Самая частая причина!**

#### Сопло слишком близко к столу

**Признаки:**
- Печатная линия намного тоньше чем нужно (может стать прозрачной)
- Излишки пластика выдавливаются вверх по краям (шероховатая поверхность, волнистые узоры)
- Филамент отрывается от стола когда печатается следующая линия
- Готовые детали очень сложно удалить со стола

**Что происходит:**
- Недостаточно места между соплом и столом для правильной экструзии
- Излишки пластика выдавливаются вверх между линиями или не экструдируются вообще (создаёт давление в HotEnd)
- Излишки пластика могут быть подняты соплом и капнуть обратно на деталь позже (особенно для PETG)

**Решение:**
- Увеличить расстояние Z-offset (поднять сопло)
- Использовать более тонкую бумагу для калибровки

#### Сопло слишком далеко от стола

**Признаки:**
- Видны отдельные линии с заметными зазорами между ними
- Линии первого слоя не касаются друг друга
- Зазоры где линии заполнения не касаются контуров стен
- Форма экструдированной линии очень круглая (правильная - слегка приплюснутая, как блинчик)
- Линии едва держатся на платформе
- Деталь отлипает или коробится

**Что происходит:**
- Недостаточно пластика экструдируется чтобы заполнить пространство
- Линии не соединяются с соседями
- Плохая адгезия → warping или отлипание

**Решение:**
- Уменьшить расстояние Z-offset (опустить сопло)
- Использовать более толстую бумагу для калибровки
- Откалибровать Z-offset (листок бумаги между соплом и столом)
- Лёгкое сопротивление при движении бумаги
- Первый слой должен быть немного прижат (0.1–0.2 мм высота)

### 2. Грязная платформа

Жир, пыль, остатки клея → плохая адгезия

**Решение:**
1. Протереть спиртом (изопропиловый спирт 99%)
2. Мыльная вода для сильных загрязнений
3. Избегать касания платформы руками

### 3. Низкая температура платформы

Пластик не успевает прилипнуть к холодному столу

**Рекомендуемые температуры:**

| Материал | Температура стола |
|----------|------------------|
| PLA | 50–60°C |
| PETG | 60–80°C |
| ABS | 100–110°C |
| Nylon | 80–100°C |
| TPU | 50–60°C |

**Решение:** Увеличить температуру стола на 10°C

### 4. Неподходящая поверхность платформы

Гладкое стекло → плохая адгезия для некоторых материалов

**Решение:** Использовать подходящую поверхность:
- **PEI** — универсально (PLA, PETG, ABS)
- **BuildTak** — хорошо для большинства материалов
- **Стекло + клей-карандаш** — для PETG
- **Каптоновая лента** — для ABS

### 5. Слишком высокая скорость первого слоя

Пластик не успевает прилипнуть

**Решение:**
- Скорость первого слоя: 15–25 mm/s (медленно!)
- Остальные слои могут быть быстрее

### 6. Низкая температура сопла для первого слоя

Пластик недостаточно жидкий

**Решение:**
- Первый слой: температура сопла +5–10°C от обычной
- Пример: PLA: 210°C первый слой, 200°C остальные

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Откалибровать Z-offset**
   - Листок бумаги между соплом и столом
   - Лёгкое сопротивление
   - Проверить все углы стола (leveling)

2. **Очистить платформу**
   - Спирт (99% изопропиловый)
   - Мыльная вода при необходимости

3. **Проверить температуру стола**
   - PLA: 60°C
   - PETG: 80°C
   - ABS: 105°C

### 🟡 Важно (если не помогло):

4. **Добавить Brim**
   - Ширина: 5–10 мм
   - Увеличивает площадь контакта
   - Помогает удерживать углы

5. **Использовать клей-карандаш / спрей**
   - Тонкий слой на стекло
   - Улучшает адгезию
   - Легко снимается

6. **Снизить скорость первого слоя**
   - 15–25 mm/s
   - Даёт время прилипнуть

### 🟢 Дополнительно:

7. **Увеличить температуру сопла первого слоя**
   - +5–10°C от обычной температуры

8. **Использовать Raft**
   - Для очень сложных случаев
   - Создаёт большую площадь контакта

## Решения по материалам

### PLA

```
Температура стола: 60°C
Скорость первого слоя: 20 mm/s
Калибровка Z: лёгкое сопротивление бумаги
PEI или стекло + клей
```

### PETG

```
Температура стола: 80°C
Скорость первого слоя: 20 mm/s
⚠️ НЕ использовать голое стекло! (вырвет кусок)
PEI или стекло + клей-карандаш (тонкий слой!)
```

### ABS

```
Температура стола: 105°C
Скорость первого слоя: 15 mm/s
Brim обязателен
PEI или каптоновая лента
Закрытая камера (50–60°C)
```

### Nylon

```
Температура стола: 90°C
Brim обязателен
PEI поверхность
Закрытая камера
```

## Быстрая диагностика

### Первый слой не прилипает вообще
→ Калибровка Z-offset или грязная платформа

### Первый слой прилипает, но отлипает позже
→ Низкая температура стола или warping

### Углы отлипают
→ Warping (добавить Brim, повысить температуру стола)

### Нити первого слоя разъезжаются
→ Сопло слишком высоко (калибровка Z)

## Checklist перед печатью

- [ ] Платформа очищена (спирт)
- [ ] Z-offset откалиброван (листок бумаги)
- [ ] Температура стола правильная
- [ ] Скорость первого слоя медленная (15–25 mm/s)
- [ ] Brim добавлен (если нужно)
- [ ] Клей-карандаш нанесён (если используете стекло)
