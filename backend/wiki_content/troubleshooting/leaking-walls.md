# Просачивание через слои (Leaking Walls) - Полное руководство

## Что такое просачивание через слои?

**Просачивание через слои** — жидкость протекает через стенки FDM-печати, стенки пористые, не герметичные. Это специфическое ограничение FDM технологии — слои не сплавляются полностью.

## Признаки проблемы

- Жидкость протекает через стенки
- Пористые стенки (видны зазоры между слоями)
- Воздух проходит через стенки
- Деталь не держит давление

## Основные причины

### 1. FDM-специфика (физическое ограничение)

**Критично понимать:** FDM печать по своей природе пористая!

- Слои не сплавляются полностью
- Всегда есть микро-зазоры между слоями
- **FDM не может быть полностью герметичным!**

**Это не дефект — это особенность технологии!**

### 2. Недостаточная толщина стенок

Тонкие стенки → больше пор → больше протечек

**Решение:**
- Увеличить толщину стенок
- Минимум 3–4 слоя (для сопла 0.4 мм = 1.2–1.6 мм)
- Лучше 5–6 слоёв для герметичности

### 3. Низкий процент заполнения (Infill)

Мало заполнения → больше пустот → протечки

**Решение:**
- Увеличить заполнение до 50–100%
- Для герметичных деталей: 80–100%

### 4. Недостаточная температура

Низкая температура → слои плохо сплавляются → больше пор

**Решение:**
- Увеличить температуру на 5–10°C
- Лучшая межслойная адгезия

### 5. Материал не подходит

Некоторые материалы более пористые чем другие

**PETG:** сложнее сделать герметичным (более вязкий)
**PLA:** лучше для герметичности
**ABS:** хорошо герметичный

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Увеличить толщину стенок**
   - Минимум 3–4 слоя (1.2–1.6 мм для сопла 0.4 мм)
   - Для герметичности: 5–6 слоёв (2.0–2.4 мм)
   - Убрать infill если нужна герметичность (solid walls)

2. **Увеличить заполнение до 80–100%**
   - Больше заполнения = меньше пустот
   - Или использовать "solid infill"

3. **Использовать правильный материал**
   - PLA: хорошо для герметичности
   - ABS: хорошо
   - PETG: сложнее (более вязкий, хуже сплавление)

### 🟡 Важно:

4. **Увеличить температуру на 5–10°C**
   - Лучшее сплавление слоёв
   - Меньше пор

5. **Уменьшить высоту слоя**
   - 0.15–0.2 mm
   - Больше слоёв = меньше зазоров

6. **Постобработка (для герметичности)**
   - Эпоксидная смола (покрытие внутренних стенок)
   - Покрытие лаком/смолой
   - Химическая обработка (для ABS: парообработка ацетоном)

## Важно понимать!

### FDM НЕ может быть полностью герметичным!

**Ограничения FDM:**
- Слои всегда имеют микро-зазоры
- Полная герметичность невозможна
- Для герметичности нужна постобработка

**Альтернативы для герметичности:**
- SLA/DLP печать (смола) — может быть герметичным
- SLS печать — может быть герметичным
- Постобработка FDM деталей (смола, покрытие)

## Настройки в слайсере

### Для максимальной герметичности:

```
Wall Line Count: 5–6 (толстые стенки)
Infill: 100% (solid) или 80%+
Layer Height: 0.15–0.2 mm (тонкие слои)
Temperature: +5–10°C (лучшее сплавление)
```

### Для обычных деталей (компромисс):

```
Wall Line Count: 3–4
Infill: 30–50%
Layer Height: 0.2 mm
Temperature: оптимальная
```

## Быстрая диагностика

### Жидкость протекает через стенки
→ FDM ограничение → увеличить толщину стенок + заполнение + постобработка

### Воздух проходит
→ Увеличить толщину стенок и заполнение

### Вода протекает
→ Требуется постобработка (эпоксидка, покрытие)

## Специфика для разных материалов

### PLA
- Хорошая герметичность (относительно)
- Толстые стенки + заполнение 80%+
- Постобработка: эпоксидная смола

### ABS
- Хорошая герметичность
- Парообработка ацетоном улучшает герметичность
- Толстые стенки + заполнение

### PETG
- Сложнее для герметичности
- Более вязкий → хуже сплавление слоёв
- Требуется больше слоёв и постобработка

## Постобработка для герметичности

### 1. Эпоксидная смола (внутреннее покрытие)

1. Залить эпоксидную смолу внутрь детали
2. Покрутить чтобы покрыть стенки
3. Вылить излишки
4. Дать застыть

**Результат:** Герметичные стенки изнутри

### 2. Покрытие лаком/смолой

- Покрыть внешние и внутренние стенки
- Может уменьшить пористость

### 3. Парообработка ацетоном (для ABS)

- Сглаживает поверхность
- Может улучшить герметичность
- Только для ABS!

## Checklist

- [ ] Толщина стенок минимум 5–6 слоёв (для герметичности)
- [ ] Заполнение 80–100%
- [ ] Температура +5–10°C от оптимальной
- [ ] Высота слоя 0.15–0.2 mm
- [ ] Постобработка (для герметичности)
