# Elephant's Foot (Слоновья нога) - Полное руководство

## Что такое Elephant's Foot?

**Elephant's Foot (слоновья нога)** — дефект когда нижние слои расширяются наружу, создавая выпуклость в нижней части детали, похожую на ногу слона. Деталь внизу шире чем должно быть по модели.

## Признаки проблемы

- Нижние слои (обычно 2–5) шире чем остальная деталь
- Выпуклость внизу детали
- Нижние грани не перпендикулярны, расширяются наружу
- Первый слой размазан по столу
- Деталь сложно снять со стола (слишком прижата)

## Основные причины

### 1. Сопло слишком близко к столу (Z-offset слишком низкий)

**Самая частая причина!**

Первый слой слишком прижат к столу → излишки пластика выдавливаются в стороны

**Решение:**
- Увеличить Z-offset (поднять сопло на 0.05–0.1 мм)
- Проверить калибровку первого слоя
- Первый слой должен быть прижат, но не размазан

### 2. Температура платформы слишком высокая

Высокая температура стола → пластик остаётся мягким → деформируется под весом верхних слоёв

**Решение:**
- Снизить температуру стола на 5–10°C после первых слоёв
- Первый слой: стандартная температура
- Слои 2+: снизить на 5–10°C

**Пример для PLA:**
- Первый слой: 60°C
- Остальные: 50°C

### 3. Отсутствие охлаждения для первых слоёв

Первые слои остаются мягкими без обдува → деформируются

**Решение:**
- Включить обдув начиная со 2–3 слоя
- Первый слой: вентилятор 0%
- Слой 2+: постепенно включать (30% → 50% → 100%)

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Отрегулировать Z-offset**
   - Поднять сопло на 0.05–0.1 мм
   - Проверить калибровку первого слоя
   - Листок бумаги должен проходить с лёгким сопротивлением

2. **Снизить температуру стола после первого слоя**
   - Первый слой: нормальная температура
   - Слои 2+: снизить на 5–10°C

### 🟡 Важно:

3. **Включить обдув начиная со 2 слоя**
   - Первый слой: 0%
   - Слой 2: 30%
   - Слой 3+: 50–100%

4. **Использовать настройку "Initial Layer Horizontal Expansion" в слайсере**
   - Отрицательное значение (-0.1 до -0.3 мм)
   - Компенсирует расширение первого слоя
   - Деталь будет немного уже внизу → после деформации будет правильного размера

### 🟢 Дополнительно:

5. **Использовать "Raft"**
   - Raft принимает на себя деформацию
   - Деталь остаётся правильного размера
   - Минус: расход материала и время

## Настройки в слайсере

### Cura

```
Initial Layer Horizontal Expansion: -0.1 to -0.3 mm
Initial Layer Flow: 90–95% (меньше пластика = меньше расширение)
Bed Temperature: 60°C (first layer), 50°C (others)
Fan Speed: 0% (layer 1), 50% (layer 2), 100% (layer 3+)
```

### PrusaSlicer / OrcaSlicer

```
Elephant foot compensation: 0.1–0.3 mm
First layer temperature: нормальная
Other layers temperature: -5–10°C
Fan speed: постепенно включать со 2 слоя
```

## Быстрая диагностика

### Нижние слои расширяются равномерно по всему периметру
→ Сопло слишком близко или температура стола высокая

### Расширение только в одном месте
→ Неравномерная калибровка стола (проверить leveling)

### Деталь очень сложно снять
→ Слишком близко сопло к столу

## Специфика для разных материалов

### PLA
- Частая проблема (мягкий материал)
- Решение: Z-offset + снизить температуру стола со 2 слоя + обдув

### PETG
- Реже (более жёсткий)
- Но может быть из-за сильной адгезии
- Решение: правильный Z-offset

### ABS
- Редко (требует высокую температуру стола)
- Решение: если есть - снизить температуру стола на 5°C

## Checklist

- [ ] Z-offset правильный (сопло не слишком близко)
- [ ] Температура стола снижается после первого слоя
- [ ] Обдув включён начиная со 2 слоя
- [ ] Используется "Initial Layer Horizontal Expansion" (-0.1 до -0.3 мм)
- [ ] Стол равномерно откалиброван (leveling)
