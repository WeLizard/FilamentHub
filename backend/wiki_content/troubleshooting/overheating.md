# Перегрев (Overheating) - Полное руководство

## Что такое Overheating?

**Перегрев (overheating)** — пластик перегревается и начинает плавиться/деформироваться во время печати. Мелкие детали плавятся, углы округляются, деталь теряет форму.

## Признаки проблемы

- Мелкие детали расплавляются/деформируются
- Углы округляются (вместо острых)
- Деталь выглядит "расплавленной"
- Слои размываются
- Потеря деталей на тонких элементах

## Основные причины

### 1. Температура слишком высокая

Пластик слишком горячий → плавится от тепла сопла

**Решение:**
- Снизить температуру сопла на 5–10°C
- Найти оптимальную температуру (температурная башня)

### 2. Слишком сильный обдув НЕ включён

Для PLA обдув критичен → без него перегревается

**Решение:**
- Включить обдув для PLA (50–100%)
- Особенно важно для мелких деталей
- Вентилятор должен работать эффективно

### 3. Медленная печать

Медленная печать → сопло долго над одним местом → перегрев

**Решение:**
- Увеличить скорость печати
- Минимум 40–50 mm/s для мелких деталей
- Быстрее = меньше времени на нагрев

### 4. Маленькие детали печатаются долго

Мелкие элементы требуют много времени → перегреваются

**Решение:**
- Увеличить скорость для мелких деталей
- Использовать "Minimum Layer Time" в слайсере
- Принтер будет ждать между слоями (даст остыть)

### 5. Нет охлаждения между слоями

Слои не успевают остыть → перегреваются

**Решение:**
- Использовать "Minimum Layer Time"
- Вентилятор на максимум для PLA
- Возможно снизить температуру для мелких деталей

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Снизить температуру на 5–10°C**
   - Найти минимальную рабочую температуру
   - Температурная башня для тестирования

2. **Включить максимальный обдув (для PLA)**
   - 100% для PLA
   - Проверить что вентилятор работает
   - Очистить вентилятор от пыли

3. **Использовать "Minimum Layer Time"**
   - 5–10 секунд между слоями
   - Принтер будет ждать пока слой остынет
   - Особенно важно для мелких деталей

### 🟡 Важно:

4. **Увеличить скорость печати**
   - Минимум 40–50 mm/s
   - Меньше времени на нагрев одного места

5. **Снизить температуру для мелких деталей**
   - Некоторые слайсеры имеют "Small Feature Speed"
   - Можно снизить температуру для мелких элементов

6. **Проверить вентилятор**
   - Работает ли эффективно
   - Нет ли препятствий
   - Очистить от пыли

## Настройки в слайсере

### Cura

```
Minimum Layer Time: 5–10 seconds
Lift Head: Да (опционально, поднимает сопло между слоями)
Cooling: 100% (для PLA)
Fan Speed: 100% (для PLA)
Small Feature Speed: 50 mm/s (для мелких деталей)
```

### PrusaSlicer / OrcaSlicer

```
Minimum layer time: 5–10 seconds
Cooling: 100% (для PLA)
Fan speed: 100% (для PLA)
```

## Быстрая диагностика

### Мелкие детали расплавляются
→ Температура слишком высокая или нет обдува

### Углы округляются
→ Перегрев от сопла → снизить температуру или увеличить скорость

### Детали теряют форму
→ Нет "Minimum Layer Time" → слои не успевают остыть

## Специфика для разных материалов

### PLA

```
Температура: 190–200°C (не выше!)
Обдув: 100% (обязательно!)
Minimum Layer Time: 5–10 seconds
Скорость: 40–50 mm/s (быстрее = лучше для мелких деталей)
```

### PETG

```
Температура: 230–240°C
Обдув: 30–50% (не 100%!)
Minimum Layer Time: 5 seconds
Менее склонен к overheating
```

### ABS

```
Температура: 240–250°C
Обдув: 0% (выключен!)
Minimum Layer Time: не требуется
Перегрев редко (высокая температура стеклования)
```

## Температурная башня

Для нахождения оптимальной температуры:

1. Распечатать температурную башню
2. Проверить каждый уровень на качество
3. Выбрать минимальную температуру без under-extrusion
4. Использовать эту температуру для печати

**Температурные башни:** доступны на Thingiverse, Printables

## Checklist

- [ ] Температура снижена (найти оптимальную)
- [ ] Обдув включён на максимум (для PLA)
- [ ] "Minimum Layer Time" установлен (5–10 секунд)
- [ ] Скорость печати достаточная (40–50 mm/s)
- [ ] Вентилятор работает эффективно
