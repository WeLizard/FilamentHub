# Полосы слоев (Layer Lines / Z-banding) - Полное руководство

## Что такое Layer Lines?

**Полосы слоев (layer lines, Z-banding)** — видимые горизонтальные полосы на боковых стенках детали, повторяющиеся на определённой высоте. Слои выглядят неровными, поверхность волнистая.

## Признаки проблемы

- Горизонтальные полосы на боковых стенках
- Волнистая поверхность
- Полосы повторяются через определённые интервалы
- Неровная поверхность даже при правильной настройке

## Основные причины

### 1. Вибрации принтера

Принтер вибрирует во время печати → слои смещаются

**Причины:**
- Принтер стоит на неустойчивой поверхности
- Высокая скорость печати
- Высокое ускорение (acceleration)

**Решение:**
1. Установить принтер на твёрдую, тяжёлую поверхность
2. Использовать антивибрационные коврики/подушки
3. Снизить скорость печати на 20–30%
4. Снизить ускорение (acceleration)

### 2. Высокая высота слоя

Толстые слои → более заметные полосы

**Решение:**
- Уменьшить высоту слоя до 0.1–0.15 мм
- Более тонкие слои = более гладкая поверхность
- Для сопла 0.4 мм: оптимально 0.2 мм

### 3. Проблемы с Z-осью

- Неровная резьба винта (lead screw)
- Загрязнённый винт
- Проблемы с муфтой/подшипниками

**Решение:**
1. Очистить винт Z-оси
2. Смазать винт (литиевой смазкой)
3. Проверить муфту Z-оси
4. Проверить выравнивание винта

### 4. Дефекты филамента

- Неровный диаметр
- Загрязнения в филаменте
- Разная температура плавления

**Решение:**
- Использовать качественный филамент
- Проверить диаметр филамента
- Очистить филамент перед печатью

### 5. Резонанс (Ringing / Ghosting)

Быстрые движения → принтер "качается" → полосы на углах

**Решение:**
1. Снизить скорость печати
2. Снизить ускорение (acceleration)
3. Снизить jerk
4. Установить принтер на тяжёлую поверхность

## Решения по приоритету

### 🔴 Критично (проверить в первую очередь):

1. **Установить принтер на твёрдую поверхность**
   - Тяжёлый стол/подставка
   - Антивибрационные коврики
   - Проверить устойчивость

2. **Снизить скорость печати на 20–30%**
   - Меньше вибраций
   - Более плавное движение

3. **Снизить высоту слоя**
   - 0.1–0.15 мм для гладкости
   - Или 0.2 мм минимум

### 🟡 Важно:

4. **Очистить и смазать Z-ось**
   - Винт Z-оси (lead screw)
   - Проверить муфту
   - Смазка литиевой смазкой

5. **Снизить ускорение (acceleration)**
   - 300–500 mm/s² вместо 1000+
   - Меньше вибраций

6. **Проверить качество филамента**
   - Ровный диаметр
   - Без загрязнений

### 🟢 Дополнительно:

7. **Использовать линейные подшипники**
   - Вместо втулок
   - Более плавное движение Z-оси

8. **Откалибровать Z-шаги**
   - Проверить правильность шагов Z-оси
   - Пересчитать если нужно

## Настройки в слайсере

```
Layer Height: 0.15–0.2 mm (для гладкости)
Print Speed: 40–50 mm/s (снизить если проблема)
Travel Speed: 100–120 mm/s
Acceleration: 500 mm/s² (снизить до 300 если нужно)
Jerk: 8 mm/s (снизить до 5)
```

## Быстрая диагностика

### Полосы через равные интервалы (например каждые 8 мм)
→ Проблема с Z-осью (винт, муфта)

### Полосы на углах
→ Резонанс (ringing) → снизить скорость/ускорение

### Полосы по всей высоте равномерно
→ Высокая высота слоя или вибрации

### Полосы неровные, случайные
→ Дефекты филамента или перепады температуры

## Специфика для разных материалов

### PLA
- Меньше проблем (низкая температура)
- Решение: снизить высоту слоя, стабилизировать принтер

### PETG
- Больше проблем (более вязкий)
- Решение: снизить скорость, уменьшить высоту слоя

### ABS
- Проблемы с охлаждением могут усиливать
- Решение: закрытая камера + снизить скорость

## Checklist

- [ ] Принтер стоит на твёрдой поверхности
- [ ] Высота слоя уменьшена (0.15–0.2 мм)
- [ ] Скорость печати снижена
- [ ] Z-ось очищена и смазана
- [ ] Ускорение снижено
- [ ] Качественный филамент
