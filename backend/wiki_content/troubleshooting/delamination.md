# Расслоение (Delamination / Layer Separation) - Полное руководство

## Что такое Delamination?

**Расслоение (delamination, layer separation)** — слои детали отслаиваются друг от друга, модель становится хрупкой и легко ломается по слоям. Критическая проблема для функциональных деталей.

## Признаки проблемы

- Слои отделяются друг от друга
- Деталь легко ломается по горизонтали
- Видны зазоры между слоями
- Слои можно разделить руками (для критических случаев)
- Деталь хрупкая, легко трескается

## Основные причины

### 1. Влажный филамент

**Самая частая причина!**

Гигроскопичные материалы впитывают влагу → при печати влага испаряется → пузыри между слоями → плохая адгезия

**Критично для:**
- Nylon (очень гигроскопичный!)
- PETG
- PLA
- TPU

**Решение:**
- Обязательная сушка филамента перед печатью
- Nylon: 80°C, 8–12 часов
- PETG: 70°C, 6–8 часов
- PLA: 60°C, 4–6 часов

### 2. Низкая температура сопла

Пластик недостаточно расплавлен → слои плохо сплавляются

**Решение:**
- Увеличить температуру сопла на 10°C
- Проверить оптимальную температуру для материала

**Рекомендуемые температуры:**
- PLA: 200–210°C
- PETG: 240–250°C
- ABS: 245–255°C
- Nylon: 250–260°C

### 3. Слишком сильное охлаждение

Для ABS/Nylon/PETG обдув снижает межслойную адгезию

**Решение:**
- ABS: вентилятор ВЫКЛЮЧЕН (0%!)
- PETG: 25–50% (не 100%!)
- Nylon: 0–25%
- PLA: можно 100% (но снизить для первых слоёв)

### 4. Высота слоя слишком большая

Толстые слои → хуже сплавление между слоями

**Решение:**
- Уменьшить высоту слоя до 0.15–0.2 мм (для сопла 0.4 мм)
- Более тонкие слои = лучшая адгезия

### 5. Скорость печати слишком высокая

Быстрая печать → пластик не успевает сплавиться со слоем ниже

**Решение:**
- Снизить скорость печати на 20–30%
- Особенно для ABS, Nylon, PC

## Решения по приоритету

### 🔴 Критично (сделать в первую очередь):

1. **Просушить филамент**
   - Обязательно для гигроскопичных материалов!
   - Nylon: 80°C, 12 часов
   - PETG: 70°C, 8 часов
   - PLA: 60°C, 6 часов

2. **Увеличить температуру сопла на 10°C**
   - Самое быстрое решение (если не влага)
   - Проверить оптимальную температуру материала

3. **Отключить/снизить обдув**
   - ABS: 0% (выключить!)
   - PETG: 30–50%
   - Nylon: 0–25%

### 🟡 Важно:

4. **Уменьшить высоту слоя**
   - 0.15–0.2 мм вместо 0.3 мм
   - Лучшая адгезия между слоями

5. **Снизить скорость печати**
   - На 20–30%
   - Особенно для ABS, Nylon, PC

6. **Хранить филамент правильно**
   - Герметичный пакет + силикагель
   - Немедленно после сушки в принтер

## Решения по материалам

### PLA

```
Температура: 200–210°C
Обдув: 50–100% (после первых слоёв)
Сушка: 60°C, 4–6ч (если влажный)
Высота слоя: 0.2 мм
```

### PETG

```
Температура: 240–250°C
Обдув: 30–50% (НЕ 100%!)
Сушка: 70°C, 6–8ч (если влажный)
Высота слоя: 0.2 мм
```

### ABS

```
Температура: 245–255°C
Обдув: 0% (ВЫКЛЮЧЕН!)
Сушка: 80°C, 2–4ч (опционально)
Высота слоя: 0.2 мм
Закрытая камера: 50–60°C
```

### Nylon

```
Температура: 250–260°C
Обдув: 0–25%
Сушка: 80°C, 8–12ч (ОБЯЗАТЕЛЬНО!)
Высота слоя: 0.2 мм
Закрытая камера: 40–50°C
```

## Быстрая диагностика

### Расслоение по всей детали
→ Влажный филамент или низкая температура

### Расслоение только в одном месте
→ Сквозняк или локальное охлаждение

### Расслоение сразу после начала печати
→ Низкая температура или высокая скорость

### Расслоение через несколько часов печати
→ Влажный филамент или перегрев/переохлаждение

## Хранение филамента

### Для предотвращения расслоения:

1. **Герметичное хранение**
   - Пакет с зип-локом
   - Силикагель внутри
   - Хранить в сухом месте

2. **Сушка перед печатью**
   - Если филамент хрустит → сушка обязательна
   - Сразу из сушилки в принтер

3. **Вакуумное хранение**
   - Для Nylon идеально
   - Предотвращает впитывание влаги

## Checklist

- [ ] Филамент просушен (если гигроскопичный)
- [ ] Температура сопла правильная (+10°C если нужно)
- [ ] Обдув отключён/снижен (для ABS/PETG/Nylon)
- [ ] Высота слоя не слишком большая (0.2 мм)
- [ ] Скорость печати не слишком высокая
- [ ] Филамент хранится в герметичном пакете
