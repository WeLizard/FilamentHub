# Трещины и разделение слоев (Cracking) - Полное руководство

## Что такое Cracking?

**Трещины (cracking) и разделение слоев** — деталь трескается, слои разделяются после печати или во время использования. Похоже на delamination, но может происходить позже.

## Признаки проблемы

- Трещины на детали после печати
- Слои разделяются (особенно вертикально)
- Деталь легко ломается
- Трещины появляются во время использования
- Хрупкость

## Основные причины

### 1. Внутреннее напряжение (Internal Stress)

**Самая частая причина!**

- Пластик остывает неравномерно
- Создаётся напряжение между слоями
- Деталь трескается при остывании или использовании

**Причины напряжения:**
- Слишком быстрое охлаждение
- Резкие перепады температуры
- Толстые стенки остывают неравномерно

**Решение:**
- Медленное охлаждение (закрытая камера для ABS)
- Постепенное снижение температуры
- Уменьшить обдув для ABS/PETG

### 2. Влажный филамент

Влажность → плохое сплавление слоёв → трещины

**Решение:**
- Просушить филамент (см. delamination)
- Хранить в герметичном пакете

### 3. Низкая температура печати

Пластик недостаточно сплавляется → слабая адгезия между слоями

**Решение:**
- Увеличить температуру на 10°C
- Улучшить межслойную адгезию

### 4. Слишком сильное охлаждение

Резкое охлаждение → напряжение → трещины

**Решение:**
- Отключить/снизить обдув для ABS
- Использовать закрытую камеру для ABS
- Медленное охлаждение

### 5. Механическое напряжение

Деталь используется не по назначению → трещины

**Решение:**
- Правильный материал для применения
- Достаточная толщина стенок
- Правильная ориентация печати

## Решения по приоритету

### 🔴 Критично:

1. **Контроль температуры охлаждения**
   - Закрытая камера для ABS (50–60°C)
   - Снизить обдув для ABS/PETG
   - Медленное охлаждение

2. **Увеличить температуру печати**
   - +10°C от оптимальной
   - Лучшая межслойная адгезия

3. **Просушить филамент**
   - Особенно для гигроскопичных материалов
   - См. статью "Delamination"

### 🟡 Важно:

4. **Использовать правильный материал**
   - ABS: склонен к трещинам (нужна закрытая камера)
   - PETG: меньше проблем
   - PLA: меньше проблем (но хрупкий)

5. **Правильная ориентация печати**
   - Избегать длинных вертикальных стенок
   - Ориентация важна для прочности

6. **Достаточная толщина стенок**
   - Минимум 3–4 слоя
   - Тонкие стенки легче трескаются

## Настройки в слайсере

### Для ABS (закрытая камера):

```
Temperature: 245–255°C
Bed Temperature: 90–100°C
Fan Speed: 0% (выключен!)
Chamber Temperature: 50–60°C (если возможно)
Layer Height: 0.2 mm
```

### Для PETG:

```
Temperature: 240–250°C
Bed Temperature: 70–80°C
Fan Speed: 30–50% (не 100%!)
Layer Height: 0.2 mm
```

## Быстрая диагностика

### Трещины сразу после печати
→ Внутреннее напряжение или быстрое охлаждение

### Трещины появляются позже
→ Влажный филамент или механическое напряжение

### Трещины вертикальные (между слоями)
→ Delamination → см. статью "Delamination"

### Трещины горизонтальные
→ Механическое напряжение или хрупкость материала

## Специфика для разных материалов

### ABS

```
⚠️ Склонен к трещинам!
Требуется закрытая камера (50–60°C)
Обдув: 0% (выключен!)
Медленное охлаждение
Температура: 245–255°C
```

### PETG

```
Меньше проблем чем ABS
Обдув: 30–50% (не 100%!)
Температура: 240–250°C
```

### PLA

```
Меньше проблем с трещинами
Но хрупкий (легко ломается)
Обдув: 100%
Температура: 200–210°C
```

## Закрытая камера для ABS

### Почему нужно:

- Предотвращает быстрое охлаждение
- Снижает внутреннее напряжение
- Предотвращает трещины

### Как сделать:

1. Закрыть принтер (корпус или экран)
2. Подогреть камеру (50–60°C)
3. Медленное охлаждение после печати

## Checklist

- [ ] Контроль охлаждения (закрытая камера для ABS)
- [ ] Обдув отключён/снижен (для ABS)
- [ ] Температура увеличена (+10°C если нужно)
- [ ] Филамент просушен
- [ ] Правильный материал для применения
- [ ] Достаточная толщина стенок
