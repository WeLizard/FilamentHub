# Забитое сопло (Clogged Nozzle) - Полное руководство

## Что такое забитое сопло?

**Забитое сопло** — сопло частично или полностью заблокировано, пластик не может выйти или выходит неравномерно. Приводит к under-extrusion или полной остановке экструзии.

## Признаки проблемы

- Пластик не выходит из сопла
- Прерывистая экструзия
- Under-extrusion (тонкие стенки, зазоры)
- Щелчки экструдера (не может продавить пластик)
- Пластик выходит криво (не прямо вниз)

## Основные причины

### 1. Остатки старого пластика

Разные материалы смешиваются → забивают сопло

**Причины:**
- Смена материала без очистки
- Остатки высокотемпературного материала (PETG, ABS) в сопле
- Застывшие остатки пластика

**Решение:**
- Прочистить сопло (cold pull)
- Использовать "чистящий филамент" (nylon, очищающий PLA)
- Прогреть сопло до высокой температуры и прочистить

### 2. Частицы мусора в филаменте

- Пыль, волокна, металлические частицы
- Особенно в дешёвом филаменте
- Композитные материалы (carbon fiber, wood)

**Решение:**
- Использовать качественный филамент
- Фильтр для филамента (опционально)
- Для композитов: стальное сопло (не латунное)

### 3. Чрезмерное обугливание

Высокая температура → пластик обугливается → забивает сопло

**Особенно:**
- PLA при слишком высокой температуре (> 230°C)
- Долгая печать на высокой температуре

**Решение:**
- Снизить температуру
- Не оставлять принтер включенным без печати

### 4. PTFE трубка деградировала (боуден)

PTFE трубка в HotEnd разрушается → частицы забивают сопло

**Признаки:**
- Запах при высоких температурах
- PTFE становится коричневым
- Проблемы появляются после долгой печати

**Решение:**
- Заменить PTFE трубку
- Или использовать all-metal HotEnd (для высоких температур)

## Решения (способы очистки)

### 🔴 Способ 1: Cold Pull (Холодная очистка)

**Самый эффективный способ!**

1. Нагреть сопло до рабочей температуры
2. Вставить филамент и выдавить немного
3. Охладить сопло до 90–100°C (для PLA) или 150°C (для PETG)
4. Потянуть филамент вниз (он вытянет застывший пластик)
5. Повторить 3–5 раз до чистоты

**Для разных материалов:**
- PLA: 90–100°C
- PETG: 150°C
- ABS: 120°C

### 🟡 Способ 2: Иголка

1. Нагреть сопло до рабочей температуры
2. Использовать иглу 0.4 мм (для сопла 0.4 мм)
3. Аккуратно прочистить сопло изнутри
4. ⚠️ Осторожно, не поцарапать сопло!

### 🟢 Способ 3: Ацетон (для PLA/ABS)

1. Снять сопло
2. Замочить в ацетоне на несколько часов
3. Очистить щёткой
4. Промыть и высушить

**⚠️ Работать в хорошо проветриваемом помещении!**

### Способ 4: Замена сопла

Если не помогает очистка:
- Проще заменить сопло (недорого)
- Особенно для латунных сопел

## Профилактика

### 1. Правильная смена материала

1. Выдавить весь старый материал
2. Использовать "чистящий филамент" (опционально)
3. Cold pull для полной очистки

### 2. Использовать качественный филамент

- Избегать дешёвого филамента с мусором
- Хранить в чистом месте

### 3. Правильная температура

- Не перегревать материал
- Использовать оптимальную температуру

### 4. Для композитных материалов

- Использовать стальное сопло (не латунное)
- Увеличить диаметр сопла (0.6 мм вместо 0.4 мм)

## Быстрая диагностика

### Пластик вообще не выходит
→ Полностью забито → cold pull или замена

### Прерывистая экструзия
→ Частично забито → cold pull

### Пластик выходит криво
→ Забито частично или повреждено сопло

### Щелчки экструдера
→ Не может продавить → забито сопло

## Checklist очистки

- [ ] Попробовать cold pull (3–5 раз)
- [ ] Если не помогает → иголка 0.4 мм
- [ ] Если не помогает → замочить в ацетоне (для PLA/ABS)
- [ ] Если не помогает → заменить сопло
- [ ] Проверить PTFE трубку (если боуден)

## Замена сопла

1. Нагреть HotEnd до рабочей температуры
2. Открутить старое сопло
3. Зачистить резьбу
4. Установить новое сопло (затянуть при температуре!)
5. Проверить экструзию

**⚠️ Затягивать сопло при температуре (горячее), иначе будет протечка!**
