# Первая 3D-печать: Пошаговое руководство

Поздравляем с покупкой 3D-принтера! Это руководство поможет вам напечатать первую деталь и избежать типичных ошибок новичков.

## Что вам понадобится

### Оборудование:
- ✅ 3D-принтер (собранный и подключенный)
- ✅ Филамент (рекомендуем PLA для начала)
- ✅ Компьютер с установленным слайсером
- ✅ SD-карта или USB-кабель для передачи файлов

### Программное обеспечение:
- **Слайсер** (выберите один):
  - [UltiMaker Cura](https://ultimaker.com/software/ultimaker-cura) (бесплатный, простой)
  - [PrusaSlicer](https://www.prusa3d.com/page/prusaslicer_424/) (бесплатный, продвинутый)
  - [OrcaSlicer](https://github.com/SoftFever/OrcaSlicer) (форк PrusaSlicer с FilamentHub интеграцией!)

### Материалы:
- **PLA филамент** — лучший выбор для начинающих:
  - Легко печатается
  - Не требует подогрева стола (но лучше с ним)
  - Не токсичен, без запаха
  - Доступен во всех цветах

## Шаг 1: Подготовка принтера

### 1.1 Распаковка и сборка

Если принтер ещё не собран:
1. Следуйте инструкции производителя
2. Проверьте все соединения (провода, ремни)
3. Установите катушку филамента на держатель

### 1.2 Калибровка стола (Bed Leveling)

**Самый важный шаг!** Неправильная калибровка → провал печати.

**Автоматическая калибровка (ABL - Auto Bed Leveling):**
1. В меню принтера выберите "Auto Home" (принтер вернётся в начало координат)
2. Выберите "Auto Bed Leveling" или "ABL"
3. Принтер прощупает несколько точек стола
4. Готово!

**Ручная калибровка (Manual Leveling):**
1. Подогрейте стол до 60°C (команда "Preheat PLA")
2. Выберите "Auto Home"
3. Отключите шаговики ("Disable Steppers")
4. Возьмите лист бумаги A4
5. Перемещайте сопло в 4 угла стола
6. В каждом углу:
   - Подложите бумагу под сопло
   - Крутите винт под столом
   - Бумага должна двигаться с лёгким сопротивлением
   - **Не слишком свободно** (зазор большой → первый слой не прилипнет)
   - **Не слишком туго** (сопло царапает → размазывает пластик)
7. Повторите для всех 4 углов + центр

**Видео-гайд:** ищите "bed leveling tutorial" на YouTube

### 1.3 Загрузка филамента

1. **Подогрейте сопло:**
   - Меню → Preheat → PLA (190–200°C)
   - Подождите 2–3 минуты

2. **Обрежьте филамент под углом 45°:**
   - Острый кончик легче войдёт в экструдер

3. **Вставьте филамент:**
   - Откройте зажим экструдера (если есть)
   - Вставьте филамент в отверстие
   - Протолкните до сопла (почувствуете сопротивление)
   - Нажмите "Load Filament" или вручную продавите

4. **Дождитесь выдавливания:**
   - Из сопла должен пойти пластик
   - Первые 5–10 см могут быть грязными (старый пластик)
   - Дождитесь чистого цвета

## Шаг 2: Выбор модели для печати

### Рекомендации для первой печати:

**Идеальные модели:**
- ✅ **Калибровочный кубик** (20×20×20 мм)
- ✅ **Benchy** (корабль-тест, 30 минут)
- ✅ **XYZ калибровочный куб**
- ✅ **Температурная башня**

**НЕ печатайте сразу:**
- ❌ Большие модели (> 50 мм)
- ❌ Мелкие детали (< 10 мм)
- ❌ Модели с нависами > 60°
- ❌ Модели без поддержек

### Где скачать модели:

1. **Thingiverse** (thingiverse.com)
   - Миллионы бесплатных моделей
   - Фильтр по популярности

2. **Printables** (printables.com)
   - От Prusa Research
   - Высокое качество

3. **Cults3D** (cults3d.com)
   - Платные и бесплатные
   - Дизайнерские модели

4. **Встроенные в слайсер:**
   - Cura: Help → Show Examples
   - PrusaSlicer: Add → Default objects

**Скачайте:** "3DBenchy.stl" (поиск в Thingiverse)

## Шаг 3: Подготовка в слайсере

### 3.1 Импорт модели

1. Откройте Cura/PrusaSlicer/OrcaSlicer
2. **Добавьте принтер:**
   - Settings → Printer → Add Printer
   - Выберите модель из списка (или Custom FDM)
   - Укажите размер стола, тип сопла

3. **Импортируйте STL:**
   - File → Open / Import
   - Или перетащите STL в окно

4. **Проверьте ориентацию:**
   - Модель должна лежать на столе (не в воздухе)
   - При необходимости поверните (кнопки Rotate)

### 3.2 Настройки для первой печати (PLA)

**Базовые параметры:**

```
Материал: PLA
Качество: 0.2 мм (Standard Quality)
Заполнение (Infill): 20%
Поддержки (Supports): НЕТ (для Benchy не нужны)
Адгезия (Adhesion): Brim (5 мм)

Температуры:
  - Сопло: 200°C
  - Стол: 60°C

Скорости:
  - Печать: 50 mm/s
  - Первый слой: 20 mm/s
  - Перемещение: 150 mm/s

Ретракт:
  - Расстояние: 5 мм (direct) или 6 мм (bowden)
  - Скорость: 45 mm/s
```

### 3.3 Слайсинг

1. Нажмите **"Slice"** (внизу справа)
2. Проверьте:
   - ✅ **Время печати:** 30–60 минут (для Benchy)
   - ✅ **Расход филамента:** 10–20 грамм
   - ✅ **Слои:** 50–100 слоёв

3. **Превью:**
   - Нажмите "Preview"
   - Прокрутите слои (ползунок справа)
   - Проверьте, что модель не в воздухе
   - Проверьте Brim (поля вокруг модели)

4. **Сохранение:**
   - Нажмите "Save to Disk" или "Save to SD"
   - Имя файла: benchy_pla_200c.gcode

## Шаг 4: Запуск печати

### 4.1 Подготовка

1. **Очистите стол:**
   - Протрите изопропанолом (70–90%)
   - Дождитесь высыхания (30 секунд)
   - НЕ трогайте руками (жир → плохая адгезия)

2. **Вставьте SD-карту** (если используется)
   - Или подключите USB

3. **Включите принтер** (если выключен)

### 4.2 Запуск

1. **На принтере:**
   - Меню → Print → Выберите файл benchy_pla_200c.gcode
   - Нажмите "Print"

2. **Или через USB:**
   - В Cura/PrusaSlicer нажмите "Print via USB"

### 4.3 Наблюдение за первым слоем (КРИТИЧНО!)

**Первый слой = 90% успеха печати**

**Первые 30 секунд:**
1. Принтер подогреет сопло и стол
2. Сопло прочертит линию по краю (prime line)
3. Начнётся печать Brim (поля)

**Проверьте первый слой:**

**✅ Хороший первый слой:**
- Линии плотно прилегают друг к другу
- Нет зазоров между линиями
- Поверхность гладкая, блестящая
- Brim прилипает к столу

**❌ Плохой первый слой:**

| Проблема | Причина | Решение |
|----------|---------|---------|
| **Линии не прилипают** | Сопло слишком высоко | Немедленно: Z-offset -0.05 мм (на принтере) |
| **Линии размазаны "блинчиком"** | Сопло слишком низко | Z-offset +0.05 мм |
| **Зазоры между линиями** | Недостаточная экструзия | Проверьте натяжение экструдера |
| **Углы отлипают** | Стол холодный/грязный | Остановите, очистите спиртом, повторите |

**Если первый слой плохой:**
1. **НЕМЕДЛЕННО остановите печать** (кнопка "Stop"/"Cancel")
2. Исправьте проблему
3. Перезапустите

**Если первый слой хороший:**
- Можете отойти
- Проверяйте каждые 10–15 минут

## Шаг 5: Завершение печати

### 5.1 Ожидание

- Не трогайте принтер во время печати
- Не открывайте дверь/окно (сквозняк → warping для ABS)
- Для PLA сквозняк ОК

### 5.2 Окончание

1. Принтер закончит печать
2. Стол и сопло начнут остывать
3. **Дождитесь остывания стола до 40°C** (для PLA)
   - Горячая деталь = мягкая = легко деформируется

### 5.3 Снятие детали

**Для PLA:**
1. Подождите, пока стол остынет до 40°C
2. Используйте шпатель/скребок
3. Аккуратно подденьте угол
4. Деталь должна легко отойти

**Для PETG:**
- **ОСТОРОЖНО!** PETG прилипает намертво
- Не используйте голое стекло (вырвет кусок)
- PEI/BuildTak → легко снимается

**Для ABS:**
- Дождитесь полного остывания (до комнатной температуры)
- Иначе деформируется

### 5.4 Постобработка

1. **Удалите Brim:**
   - Кусачками или ножом
   - Осторожно, не повредите деталь

2. **Проверьте качество:**
   - Слои ровные?
   - Нет зазоров?
   - Мосты не провисли?

## Частые проблемы первой печати

| Проблема | Причина | Решение |
|----------|---------|---------|
| **Первый слой не прилипает** | Плохая калибровка стола | Перекалибруйте, очистите спиртом |
| **Углы загибаются (warping)** | Нет подогрева стола | Включите подогрев 60°C, добавьте Brim |
| **Волосатость (stringing)** | Высокая температура | Снизьте на 5°C, проверьте ретракт |
| **Слои разваливаются** | Низкая температура | Повысьте на 5°C |
| **Сопло забилось** | Старый/грязный филамент | Прочистите иглой, сделайте "cold pull" |

## Checklist первой печати

Перед запуском проверьте:

- [ ] Стол откалиброван (бумага с лёгким сопротивлением)
- [ ] Стол чистый (протёрт спиртом)
- [ ] Филамент загружен (идёт из сопла)
- [ ] Модель импортирована в слайсер
- [ ] Настройки PLA: 200°C/60°C
- [ ] Brim включён
- [ ] Файл сохранён на SD-карту
- [ ] Готовы наблюдать за первым слоем!
