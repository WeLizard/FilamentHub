# Необходимые инструменты и расходные материалы для 3D-печати

При работе с 3D-принтером вам понадобятся различные инструменты и расходные материалы. Этот гайд поможет собрать базовый набор для комфортной работы.

## Клей для 3D-печати

**Что это:** Специальные адгезивы для улучшения прилипания первого слоя к столу.

**Варианты:**

1. **Клей на основе PVP (Поливинилпирролидон)**
   - Находится в "Энтеродезе" из аптеки
   - Или в клеях-карандашах, где указано PVP (например, "Каляка-маляка")
   - Работает для PLA, PETG

2. **Готовые решения:**
   - 3D Click (спрей)
   - 3D SOLID GLUE (карандаш)

**Применение:** Наносится тонким слоем на поверхность стола перед печатью для улучшения адгезии, особенно для сложных материалов (PETG, ABS).

---

## Маркер для нанесения клея (опционально)

**Что это:** Пустой многоразовый маркер для заправки клеем.

**Характеристики:**
- Ширина пера: 50 мм
- Помповая система
- Для всех типов краски/клея
- Многоразовый

**Зачем нужен:**
- Равномерное нанесение клея на стол
- Альтернатива спрею (который может попасть на механику принтера и повлиять на её работу)
- Контролируемое нанесение без излишков

**Альтернатива:** Можно использовать обычный клей-карандаш или распылитель, но с осторожностью.

---

## Бокорезы (Side Cutters)

**Что это:** Инструмент для обрезки филамента.

![Бокорезы](../uploads/side-cutters.jpg)

**Требования:**
- Прямая режущая кромка
- Для точных работ
- Должны легко резать PLA, PETG, ABS

**Применение:**
- Обрезка филамента при загрузке/выгрузке
- Удаление поддержек (supports)
- Обрезка "волосков" после печати
- Работа с готовыми изделиями

**Рекомендация:** Имеют под рукой постоянно - используются очень часто!

---

## Фаскосниматель (Chamfer Tool)

**Что это:** Инструмент для обработки краёв готовых изделий.

![Фаскосниматель](../uploads/chamfer-tool.jpg)

**Применение:**
- Удаление заусенцев
- Скругление острых краёв
- Обработка отверстий
- Улучшение внешнего вида готовых деталей

**Совместимость:** BS 1010, BS 1012, BS 1018, BS 2010, BS 3010

**Когда нужен:** Для финальной постобработки функциональных и декоративных деталей.

---

## PTFE Трубка (Bowden)

**Что это:** Фторопластовая трубка для передачи филамента от экструдера к соплу.

![PTFE трубка](../uploads/ptfe-tube.jpg)

**Характеристики:**
- **Внутренний диаметр:** 1.9 мм (или 2.0 мм)
- **Внешний диаметр:** 4.0 мм
- **Длина:** 1 м (стандартная)
- **Материал:** PTFE (тефлон)

**Зачем иметь запас:**
- Фитинги (соединители) изнашивают внутреннюю поверхность трубки
- Периодически требуется подрезка (обрезать изношенный конец)
- Со временем трубка может забиться или деформироваться

**Рекомендация:** Держите 1-2 метра в запасе на случай замены.

---

## Фитинги для трубки

**Что это:** Соединители для PTFE трубки в экструдере и хот-энде.

![Фитинги](../uploads/fittings.jpg)

**Типы:**
- **PC6-01** - для трубки 6 мм
- **PC4-01** - для трубки 4 мм
- **PC4-M6** - для трубки 4 мм с резьбой M6

**Особенности:**
- Внутри есть зубчики (зажимы), которые удерживают трубку
- Зубчики могут изредка вырываться при демонтаже
- Пока новые - не требуется замена, но стоит иметь запасные

**Рекомендация:** 2-4 запасных фитинга разных типов на случай поломки.

---

## Сопла (Nozzles)

**Что это:** Основной расходный материал 3D-принтера.

![Сопла](../uploads/nozzles.jpg)

**Размеры:**
- **0.2 мм** - для высокоточных деталей, медленная печать
- **0.3 мм** - баланс качества и скорости
- **0.4 мм** - стандартный размер, самый популярный ⭐
- **0.5 мм** - быстрая печать, большие детали
- **0.6 мм** - очень быстрая печать, толстые слои
- **0.8 мм** - для крупных моделей
- **1.0 мм** - для очень крупных деталей

**Материалы:**
- **Латунь** - стандарт, для большинства материалов (PLA, PETG, ABS)
- **Нержавеющая сталь** - для абразивных материалов (карбон, стекловолокно)
- **Закалённая сталь** - максимальная износостойкость

**Рекомендация:**
- Начните с 0.4 мм - это самый универсальный размер
- Имейте 2-3 запасных сопла (на случай засора или износа)
- Выбирайте размер в зависимости от требуемого качества и скорости

---

## Термисторы (Температурные датчики)

**Что это:** Датчики температуры для хот-энда и стола.

![Термистор](../uploads/thermistor.jpg)

**Типы:**
- **ATC Semitec 104NT** - популярный тип
- В виде капли (droplet)
- В гильзе (sleeve) - вставляется в термоблок
- С резьбой - вкручивается в термоблок

**Зачем иметь запас:**
- Термисторы часто ломаются (обрыв провода, перегрев)
- Без исправного термистора принтер не может контролировать температуру
- Критически важный компонент безопасности

**Рекомендация:** Храните 1-2 запасных термистора. Обязательно узнайте тип, используемый в вашем принтере!

---

## Нагревательные элементы (Heater Cartridges)

**Что это:** Нагреватели для хот-энда.

![Нагревательный элемент](../uploads/heater.jpg)

**Характеристики:**
- **Напряжение:** 12V или 24V (зависит от вашего принтера)
- **Мощность:** 40W или 50W (чаще всего)
- **Размер:** обычно 6.35 мм (M6) диаметр

**Важно:**
- **Обязательно узнайте напряжение вашего принтера!** (12V или 24V)
- Подключение 24V нагревателя к 12V системе = слабый нагрев
- Подключение 12V нагревателя к 24V системе = **сгорит!** ⚠️

**Рекомендация:**
- Имейте 1 запасной нагревательный элемент
- Подпишите на нём напряжение и мощность (например, "12V/40W")
- Не путайте с термистором (термистор - датчик, нагреватель - греет)

## Штангенциркуль (Caliper)

**Что это:** Инструмент для точных измерений.

![Штангенциркуль](../uploads/caliper.jpg)

**Типы:**
- **Электронный (цифровой)** - точность 0.01-0.02 мм, удобный
- **Механический** - точность 0.05-0.1 мм
- **Пластиковый электронный** (например, из Fix Price) - бюджетный вариант, точность ~0.1 мм

**Применение:**
- Измерение диаметра сопла
- Проверка калибровки принтера (размеры напечатанных деталей)
- Измерение толщины слоёв
- Измерение филамента (проверка диаметра)

**Рекомендация:**
- Для начала можно использовать пластиковый за 200-300₽
- Для серьёзной работы лучше металлический электронный (точность 0.01-0.02 мм)
- Точность критична для калибровки принтера!

## Поверхность стола (Build Surface)

**Что это:** Платформа, на которой печатаются детали.

![Поверхность стола](../uploads/build-surface.jpg)

**Типы:**

### 1. Стекло (Glass)
- **Плюсы:** Гладкая поверхность, дешёвая
- **Минусы:** Требует клей/каптон для адгезии, хрупкая

### 2. PEI (Полиэтиленимид)
- **Плюсы:** Отличная адгезия, легко снимать детали, долговечная
- **Виды:**
  - Гладкая (smooth) - глянцевая поверхность
  - Текстурированная (textured) - передаёт узор на первый слой, красиво

### 3. Магнитная основа + PEI лист
- **Плюсы:** 
  - PEI лист легко снимается для удаления деталей
  - Удобно менять поверхность
  - Текстурированные PEI листы выглядят отлично

**Рекомендация:**
- Для новичков: PEI на магнитной основе (удобно)
- Для качества: PEI текстурированная (красивый первый слой)
- Для экономии: Стекло + клей (но менее удобно)

## Набор инструментов для обслуживания

**Что это:** Базовые инструменты для обслуживания принтера.

![Набор инструментов](../uploads/printer-tools.jpg)

**Минимальный набор:**

1. **Щётка** - для очистки стола и механизмов
2. **Ключ для сопла** - специальный ключ для замены сопла
3. **Шестигранные ключи (Hex keys)** - размеры от 1.5 до 8 мм
   - 1.5, 2, 2.5, 3 мм - для мелких винтов
   - 4, 5, 6, 8 мм - для крупных соединений
4. **Отвёртки** - крестообразные и плоские
5. **Иглы для прочистки сопла** - 0.2-0.4 мм

**Применение:**
- Регулярное обслуживание (очистка, смазка)
- Замена компонентов (сопла, термисторы)
- Калибровка и настройка
- Ремонт при поломках

**Рекомендация:** Соберите полный набор заранее - при поломке не будет времени искать инструменты!

## Набор скальпелей (Hobby Knives)

**Что это:** Тонкие ножи для точной обрезки и обработки деталей.

![Набор скальпелей](../uploads/scalpel-set.jpg)

**Типы:**
- Набор с разными лезвиями (удобно иметь разные формы)
- Одиночный качественный скальпель

**Применение:**
- Удаление мелких дефектов печати
- Обрезка поддержек в труднодоступных местах
- Тонкая обработка поверхностей
- Удаление "волосков" и стрингинга

**Рекомендация:** Можно использовать набор из Galamart или другой бюджетный вариант. Достаточно одного качественного скальпеля, но набор удобнее.

## Резьбовые втулки (Threaded Inserts)

**Что это:** Металлические втулки с резьбой, которые вплавляются в пластик.

![Резьбовые втулки](../uploads/threaded-inserts.jpg)

**Размеры:**
- **M3** - для мелких креплений
- **M4** - стандартный размер
- M5, M6 - для крупных креплений

**Важно:**
- **Обязательно с косыми зубцами** (slanted teeth) - держат лучше
- **НЕ прямые зубцы** - это для литья под давлением, не для 3D-печати!

**Применение:**
- Крепление деталей винтами
- Усиление резьбовых отверстий в пластике
- Создание разборных соединений
- Повышение прочности резьбы в напечатанных деталях

**Установка:**
- Разогреть втулку (паяльником или специальным инструментом)
- Вплавить в отверстие в пластике
- Дать остыть

**Рекомендация:** Покупайте несколько наборов M3 и M4 - используются часто для функциональных деталей.

## 3D Ручка

**Что это:** Портативный 3D-принтер в виде ручки для ручной "печати" пластиком.

![3D ручка](../uploads/3d-pen.jpg)

**Зачем нужна:**
- **Подправка деталей** - когда не хочется перепечатывать всю деталь
- **Ремонт** - заполнение трещин, сколов
- **Добавление элементов** - мелкие детали, усиления
- **Художественная доработка** - украшения, надписи

**Применение:**
- Ремонт крупных деталей (перепечатывать целиком дорого по времени/материалу)
- Заполнение зазоров
- Усиление слабых мест

**Дополнительно:**
- **Паяльник с утюжком** - для более серьёзного ремонта пластиковых деталей (продвинутый уровень)

**Рекомендация:** Полезный инструмент, но не критичный. Покупайте если планируете часто дорабатывать/ремонтировать детали.

## Цианакрилатный клей (Суперклей) с активатором

**Преимущества:**
- Мгновенное схватывание с активатором

- Подходит для большинства пластиков

## Грунт для пластика

**Что это:** П primer для подготовки пластиковой поверхности перед покраской.

![Грунт для пластика](../uploads/primer.jpg)

**Типы:**

1. **Акриловый грунт**
   - Базовая подготовка поверхности
   - Улучшает адгезию краски

2. **Грунт-наполнитель** (filler primer)
   - Заполняет мелкие неровности
   - Скрывает следы от слоёв печати
   - Создаёт гладкую поверхность

**Цвета:**
- Серый - универсальный
- Белый - для светлых красок
- Чёрный - для тёмных красок

**Бренды:**
- KUDO
- И другие автомобильные грунты

**Применение:**
- Перед покраской акриловыми красками
- Для сглаживания поверхности
- Для профессионального внешнего вида

**Рекомендация:** Обязательно, если планируете красить детали. Без грунта краска может отслаиваться.

## Растворители для пластика

**⚠️ ВНИМАНИЕ!** Перед использованием изучите каждый растворитель! Растворители для разных пластиков разные.

![Растворители](../uploads/solvents.jpg)

### Основные растворители:

1. **Дихлорметан (Dichloromethane, CH₂Cl₂)**
   - Для: **PETG**, **PLA**
   - ⚠️ Токсичен! Использовать в хорошо проветриваемом помещении
   - Химическая полировка поверхностей

2. **D-Limonene (D-Лимонен)**
   - Для: **HIPS**, **SBS**
   - Менее токсичен, чем дихлорметан
   - Используется для растворения HIPS поддержек

3. **Ацетон (Acetone)**
   - Для: **ABS**, **ASA**
   - Популярный растворитель для сглаживания ABS
   - ⚠️ Огнеопасен!

4. **Тетрагидрофуран (ТГФ, Tetrahydrofuran, THF)**
   - Для: **PLA**, **ABS**, **PVC**
   - Мощный растворитель
   - ⚠️ Очень токсичен!

### Таблица совместимости растворителей:

| Пластик | Обозначение | Ацетон | Дихлорметан | ТГФ | D-Лимонен | Другие |
|---------|-------------|--------|-------------|-----|----------|--------|
| **ABS** | ABS | ✅ | ✅ | ✅ | ❌ | Этилацетат, Толуол |
| **PLA** | PLA | ❌ | ✅ | ✅ | ❌ | - |
| **PETG** | PETG | ❌ | ✅ | ✅ | ❌ | - |
| **HIPS** | HIPS | ✅ | ✅ | ✅ | ✅ | Этилацетат, Толуол |
| **PVC** | PVC | ❌ | ✅ | ✅ | ❌ | Циклогексанон |
| **PC** | PC | ❌ | ✅ | ✅ | ❌ | Дихлорэтан |
| **Nylon** | PA | ❌ | ❌ | ⚠️ | ❌ | Муравьиная кислота |
| **PMMA** | Акрил | ✅ | ✅ | ✅ | ❌ | Этилацетат |

![Таблица совместимости](../uploads/solvent-compatibility-table.jpg)

### Применение растворителей:

1. **Химическая полировка** - сглаживание поверхности
2. **Склейка** - растворение поверхностного слоя для склейки
3. **Удаление поддержок** - растворение растворимых материалов (HIPS)
4. **Постобработка** - удаление слоёв, сглаживание

### ⚠️ Техника безопасности:

- Использовать в хорошо проветриваемом помещении
- Защитные перчатки
- Очки для защиты глаз
- Огнеопасно - не курить, не использовать рядом с огнём
- Хранить в недоступном для детей месте

## Термометр-гигрометр (Digital Thermometer-Hygrometer)

**Что это:** Цифровой прибор для измерения температуры и влажности в помещении.

![Термометр-гигрометр](../uploads/thermometer-hygrometer.jpg)

**Характеристики:**
- Температура: обычно 0-50°C
- Влажность: 0-100% RH
- Точность: ±1°C, ±5% RH
- Питание: батарейка (CR2032 или подобная)

**Зачем нужен:**
- **Контроль условий хранения филамента** - влажность критична!
- **Мониторинг температуры в камере принтера** - для ABS/ASA
- **Оптимизация условий печати**
- **Создание пассивной термокамеры** - контроль температуры внутри

**Применение:**
- В помещении где хранится филамент
- Внутри принтера (если есть камера)
- Для проверки условий сушки филамента

**Рекомендация:** Дешёвый и очень полезный инструмент. Помогает понять почему филамент влажный или почему детали деформируются.

## Смазочные вещества

**Что это:** Смазки для механических компонентов принтера.

![Смазки](../uploads/lubricants.jpg)

**Типы:**

1. **Автоматическая трансмиссионная жидкость (ATF)**
   - Mobil ATF 3309 и аналоги
   - Для направляющих (linear rails)
   - Для подшипников скольжения

2. **Консистентная смазка (Grease)**
   - MOLYKOTE Multilub и аналоги
   - Для подшипников качения
   - Для винтовых пар (lead screws)

**Применение:**
- **Направляющие (Linear Rails)** - плавность движения
- **Подшипники** - снижение износа
- **Винтовые пары** - точность позиционирования
- **Другие движущиеся части**

**Важно:**
- Выбирайте в зависимости от механики вашего принтера
- Не смешивайте разные типы смазок
- Периодически обновляйте (раз в 3-6 месяцев)

**Рекомендация:** Обязательно для поддержания принтера в рабочем состоянии. Без смазки механизм изнашивается быстрее.
