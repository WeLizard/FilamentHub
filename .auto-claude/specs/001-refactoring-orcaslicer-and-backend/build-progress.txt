=== CRITICAL PATTERNS ===

C++ BoostThreadWorker Pattern:
```cpp
// CORRECT - Use shared_ptr
auto job = std::make_shared<SyncJob>(coordinator);
m_worker.push(job);

// WRONG - unique_ptr will fail!
auto job = std::make_unique<SyncJob>(...); // BoostThreadWorker requires shared_ptr
```

SQLAlchemy 2.0 Async Pattern:
```python
# CORRECT
result = await db.execute(select(Model).where(...))
return result.scalar_one_or_none()

# DEPRECATED - DO NOT USE
db.query(Model).filter(...)
```
