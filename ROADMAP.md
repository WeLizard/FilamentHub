# FilamentHub - Roadmap развития проекта

## 🎯 Видение проекта
**FilamentHub** — централизованная платформа для управления материалами и настройками в 3D-печати, объединяющая производителей пластиков, пользователей и слайсеры (OrcaSlicer, SuperSlicer, Bambu Studio).

**Слоган:** "Ваш пластик уже настроен. Осталось только печатать."

---

## 📊 Текущее состояние

**Дата обновления:** 2025-11-05  
**Статус:** 
- ✅ Backend MVP ~95% завершен (все основные эндпоинты реализованы, OrcaSlicer экспорт работает, QR-коды, управление пользователями)
- 🔥 Frontend MVP ~85% завершен (UI полностью интегрирован, создание пресетов с OrcaSlicer параметрами, Brand Requests система, админ-панель)

- ✅ Управление пользователями в админ-панели (отвязка от брендов, просмотр информации о брендах)

### Существующие компоненты:
- ✅ **3dcalc** - PHP-приложение с:
  - Калькулятором стоимости печати
  - API для управления материалами
  - G-code парсером (Cura, PrusaSlicer, SuperSlicer, OrcaSlicer, Bambu Studio, Creality)
  - Базовым UI (HTML/JS/TailwindCSS)
  - MySQL базой данных

### Референсные проекты:
- 📚 **Spoolman** - Python FastAPI + React (для изучения архитектуры API и БД)
- 📚 **OrcaSlicer** - C++ слайсер (для понимания интеграции)
- 📚 **spoolman2slicer** - Python скрипт для экспорта данных в слайсеры

```python
# Основные сущности
- Brand (Производитель)
  - id, name, description, website, logo_url, verified

- Filament (Материал)
  - id, brand_id, name, material_type, color_hex

  - created_at, updated_at
  
- User (Пользователь)
  - id, email, username, role (user/brand/admin)
  - api_key (для плагинов)
  
- Review (Отзыв)
  - id, preset_id, user_id, rating, comment
  - success (boolean), photo_url
```

- **UI:** wxPython (если нужен GUI)

- **Monitoring:** Prometheus + Grafana
- **Logging:** Sentry
