## 🔍 Что происходит во время сборки:

### Этап 1: Зависимости (deps)
```
cmake ../ -G "Visual Studio 17 2022" -A x64 -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release --target deps -- -m
```
- **Время:** ~30-60 минут
- **Вмешательство:** ❌ НЕ требуется
- **Автоматически:**
  - Настраивает CMake проект для зависимостей
  - Компилирует все зависимости
  - Создает библиотеки (.lib) и DLL

### Этап 2: OrcaSlicer
```
cmake .. -G "Visual Studio 17 2022" -A x64 -DORCA_TOOLS=ON -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release --target ALL_BUILD -- -m
```
- **Время:** ~30-60 минут
- **Вмешательство:** ❌ НЕ требуется
- **Автоматически:**
  - Настраивает CMake проект для OrcaSlicer
  - Компилирует весь код OrcaSlicer
  - Создает исполняемый файл

### Этап 3: Локализация (gettext)
```
call scripts/run_gettext.bat
```
- **Время:** ~1-2 минуты
- **Вмешательство:** ❌ НЕ требуется
- **Автоматически:**
  - Обрабатывает файлы локализации (.po)
  - Создает скомпилированные файлы (.mo)

### Этап 4: Установка (install)
```
cmake --build . --target install --config Release
```
- **Время:** ~1 минута
- **Вмешательство:** ❌ НЕ требуется
- **Автоматически:**
  - Копирует файлы в `build\package\bin\`
  - Создает финальный `orca-slicer.exe`

## ⚠️ Потенциальные проблемы (но редко):

### 1. Ошибки компиляции
- **Если:** Ошибка компиляции
- **Что:** Сборка остановится с ошибкой
- **Вмешательство:** ❌ НЕ требуется (просто остановится)
- **Решение:** Утром посмотрите логи в окне CMD

### 2. Нехватка места на диске
- **Если:** Мало места (нужно ~5-10 GB)
- **Что:** Сборка может упасть
- **Вмешательство:** ❌ НЕ требуется (просто остановится)

### 3. Прерывание питания/сна ПК
- **Если:** ПК уйдет в сон или отключится питание
- **Что:** Сборка остановится
- **Вмешательство:** ❌ НЕ требуется (но сборка не завершится)
