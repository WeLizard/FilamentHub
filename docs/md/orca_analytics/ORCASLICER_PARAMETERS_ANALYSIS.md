# Анализ параметров OrcaSlicer для импорта/экспорта

## Исследование: Какие параметры действительно важны?

На основе анализа интернета, документации OrcaSlicer и реальных пользовательских профилей.

## Критически важные параметры (MUST HAVE)

### 1. Температурные настройки
**Приоритет: ВЫСОКИЙ**

- `nozzle_temperature` - температура экструдера (критично для качества)
- `nozzle_temperature_initial_layer` - температура для первого слоя
- `nozzle_temperature_range_low/high` - рабочий диапазон температур
- `hot_plate_temp`, `cool_plate_temp`, `eng_plate_temp`, `textured_plate_temp` - температуры стола
- `*_plate_temp_initial_layer` - температуры стола для первого слоя

**Почему важно:** Неправильная температура = плохое качество печати, проблемы с адгезией

### 2. Ретракция (Retraction)
**Приоритет: ВЫСОКИЙ**

- `filament_retraction_length` - длина ретракции (предотвращает оозинг)
- `filament_retraction_speed` - скорость ретракции
- `filament_deretraction_speed` - скорость де-ретракции
- `filament_retraction_minimum_travel` - минимальное расстояние для ретракции
- `filament_z_hop` - подъем Z при ретракции (важно для некоторых принтеров)

**Почему важно:** Без ретракции = нити между деталями, плохое качество поверхности

### 3. Вентиляторы (Fans)
**Приоритет: ВЫСОКИЙ**

- `fan_min_speed` - минимальная скорость вентилятора (0-100%)
- `fan_max_speed` - максимальная скорость вентилятора
- `overhang_fan_speed` - скорость вентилятора для свесов
- `overhang_fan_threshold` - порог активации вентилятора для свесов (например "25%")
- `fan_cooling_layer_time` - время охлаждения слоя

**Почему важно:** Охлаждение критично для качества деталей, предотвращения деформаций

### 4. Свойства филамента (Filament Properties)
**Приоритет: ВЫСОКИЙ**

- `filament_type` - тип материала (PLA, ABS, PETG, etc.) - ОБЯЗАТЕЛЬНО
- `filament_vendor` - производитель - ОБЯЗАТЕЛЬНО
- `filament_diameter` - диаметр (1.75 или 2.85) - ОБЯЗАТЕЛЬНО
- `filament_density` - плотность в г/см³ (для расчета веса)
- `filament_cost` - стоимость (для калькулятора)
- `default_filament_colour` - цвет филамента

**Почему важно:** Базовые свойства материала, без них профиль неполный

### 5. Коэффициент потока (Flow Ratio)
**Приоритет: СРЕДНИЙ**

- `filament_flow_ratio` - коэффициент потока (множитель, например "0.95")
- `filament_max_volumetric_speed` - максимальная объемная скорость экструзии

**Почему важно:** Компенсирует недо/переэкструзию, улучшает качество стенок

### 6. Pressure Advance (для OrcaSlicer)
**Приоритет: СРЕДНИЙ (для продвинутых)**

- `pressure_advance` - значение pressure advance (например "0.038")
- `enable_pressure_advance` - включить/выключить (0|1)
- `adaptive_pressure_advance` - адаптивный PA (0|1) - опционально

**Почему важно:** Уникальная фича OrcaSlicer, улучшает качество углов и резких деталей

## Важные, но не критичные параметры (SHOULD HAVE)

### 7. Усадка и компенсация
- `filament_shrink` - усадка филамента (например "99.8%")
- `filament_shrinkage_compensation_z` - компенсация усадки по Z (например "100%")

### 8. Дополнительные температурные
- `temperature_vitrification` - температура витрификации
- `idle_temperature` - температура простоя

### 9. Специальные режимы
- `slow_down_for_layer_cooling` - замедление для охлаждения слоя
- `slow_down_layer_time` - время слоя для замедления
- `slow_down_min_speed` - минимальная скорость при замедлении

### 10. Настройки загрузки/выгрузки (для мультитула)
- `filament_loading_speed` - скорость загрузки
- `filament_unloading_speed` - скорость выгрузки
- `filament_multitool_ramming` - ramming для мультитула

## Менее важные параметры (NICE TO HAVE)

### 11. Камера и фильтрация
- `chamber_temperature` - температура камеры
- `activate_chamber_temp_control` - контроль температуры камеры
- `activate_air_filtration` - воздушная фильтрация

### 12. G-code (пользовательские скрипты)
- `filament_start_gcode` - начальный G-code
- `filament_end_gcode` - конечный G-code

**Почему менее важно:** Очень специфично для конкретного принтера/прошивки

### 13. Scarf seam (шов)
- `filament_scarf_seam_type` - тип шва
- `filament_scarf_height/gap/length` - параметры шва

### 14. Wipe (очистка)
- `filament_wipe` - включить очистку
- `filament_wipe_distance` - расстояние очистки
- `filament_retract_before_wipe` - ретракция перед очисткой

### 15. Мультитул специфичные
- `filament_stamping_distance` - расстояние штамповки
- `filament_toolchange_delay` - задержка смены инструмента
- `filament_cooling_initial/final_speed` - охлаждение при загрузке

## Параметры, которые НЕ нужно импортировать (SKIP)

### 16. Совместимость принтеров
- `compatible_printers` - список совместимых принтеров
- `compatible_printers_condition` - условие совместимости

**Почему:** Слишком специфично для конкретного принтера, должен быть в FilamentHub принтер-специфичный

### 17. Системные/внутренние
- `type` - тип профиля (определяется автоматически)
- `from` - источник (system/user) - определяется автоматически
- `inherits` - базовый профиль (определяется по material_type)
- `version` - версия OrcaSlicer
- `instantiation` - внутренний флаг
- `setting_id`, `filament_id` - ID генерируются автоматически

### 18. Параметры, специфичные для принтера
- `filament_extruder_variant` - вариант экструдера (Direct Drive/Bowden)
- `required_nozzle_HRC` - требуемая твердость сопла
- Все параметры с префиксом `compatible_*`

## Рекомендации по приоритетам

### Уровень 1: ОБЯЗАТЕЛЬНО (для базового профиля)
1. Температуры (nozzle, bed)
2. Ретракция (length, speed)
3. Вентиляторы (min/max speed)
4. Свойства филамента (type, vendor, diameter, density)

**Всего: ~15-20 параметров**

### Уровень 2: ВАЖНО (для продвинутого профиля)
5. Flow ratio, max volumetric speed
6. Pressure advance
7. Усадка и компенсация
8. Дополнительные температурные
9. Специальные режимы охлаждения

**Всего: ~10-15 дополнительных параметров**

### Уровень 3: ОПЦИОНАЛЬНО (для полного профиля)
10. Камера и фильтрация
11. G-code скрипты
12. Scarf seam, wipe
13. Мультитул параметры

**Всего: ~20-30 дополнительных параметров**

### Уровень 4: ИГНОРИРОВАТЬ
- Совместимость принтеров
- Системные параметры
- Принтер-специфичные параметры

## Выводы для FilamentHub

### Рекомендуемый подход:

**Базовые параметры в БД (колонки):**
1. Температуры (extruder_temp, bed_temp)
2. Ретракция (retraction_length, retraction_speed)
3. Вентилятор (fan_speed)
4. Свойства филамента (уже есть в модели Filament)
5. Flow ratio (flow_rate)

**Расширенные параметры в JSON (settings_json):**
- Все остальные параметры уровней 2-3
- Позволяет хранить полный профиль OrcaSlicer
- Можно расширять без изменения БД

**При экспорте:**
- Базовые параметры → OrcaSlicer формат
- JSON параметры → добавляются к профилю
- Значения по умолчанию → заполняются из базового профиля OrcaSlicer

**При импорте:**
- Парсим JSON профиль OrcaSlicer
- Базовые параметры → колонки БД
- Остальные параметры → JSON поле

## Статистика параметров

- **Критически важных:** ~20 параметров
- **Важных:** ~15 параметров
- **Опциональных:** ~30 параметров
- **Игнорировать:** ~48 параметров

**Итого для импорта/экспорта: ~65 параметров из 113**

## Рекомендация

**Начать с базовых 20 параметров** (уровень 1) - этого достаточно для большинства пользователей.

**Постепенно добавлять расширенные** (уровни 2-3) - по запросу пользователей или для продвинутых профилей.

**Хранить все в JSON при импорте** - даже если не используем, сохраняем для будущего расширения.
