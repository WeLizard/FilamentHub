# FilamentHub Edition 2.0.0

Первый релиз OrcaSlicer с интегрированным каталогом материалов FilamentHub.

## 🎯 Что это?

**FilamentHub Edition** — это полная версия OrcaSlicer со встроенной интеграцией с каталогом материалов FilamentHub. Все функции оригинального слайсера сохранены без изменений — мы только добавили удобный доступ к каталогу материалов прямо в интерфейсе слайсера.

## ✨ Основные возможности

- **Вкладка FilamentHub** — новая вкладка прямо в главном окне OrcaSlicer
- **Встроенный каталог материалов** — просматривайте настройки, фильтруйте по принтерам и сразу импортируйте пресеты
- **Автоматическая синхронизация пресетов** — ваши настройки синхронизируются между FilamentHub и OrcaSlicer
- **Импорт одним кликом** — нашли нужные настройки? Импортируйте пресет прямо в OrcaSlicer одним кликом
- **100% функционал OrcaSlicer** — все функции оригинального слайсера сохранены

## 📦 Что включено в релиз

### Windows
- **Installer** (`OrcaSlicer-FilamentHub-2.0.0-fh-win64.exe`) — установщик для Windows 10/11 (111 MB)
- **Portable** (`OrcaSlicer-FilamentHub-2.0.0-fh-win64-portable.zip`) — портативная версия в ZIP архиве, не требует установки (140 MB)

## 🚀 Установка

### Windows (Installer)
1. Скачайте файл `OrcaSlicer-FilamentHub-2.0.0-fh-win64.exe`
2. Запустите установщик и следуйте инструкциям
3. После установки откройте OrcaSlicer — вкладка FilamentHub будет доступна в главном окне
4. Войдите в свой аккаунт FilamentHub через вкладку FilamentHub (регистрация не обязательна для просмотра каталога)

### Windows (Portable)
1. Скачайте файл `OrcaSlicer-FilamentHub-2.0.0-fh-win64-portable.zip`
2. Распакуйте архив в любую папку
3. Запустите `orca-slicer.exe` из распакованной папки
4. Вкладка FilamentHub будет доступна в главном окне

## 🔒 Безопасность

- Основана на открытом исходном коде OrcaSlicer
- Все изменения доступны в открытом репозитории
- Безопасная интеграция через WebView
- Никаких изменений в ядре слайсера
- Все коммуникации через HTTPS

## 📝 Примечания

- FilamentHub Edition может быть установлена параллельно с обычной OrcaSlicer — они не конфликтуют
- Все ваши пресеты и настройки OrcaSlicer останутся на месте
- Аккаунт FilamentHub не обязателен для просмотра каталога и импорта настроек
