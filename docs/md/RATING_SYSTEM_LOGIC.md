# Логика системы рейтингов и отзывов

## Правила создания отзыва:

### 1. Условие для создания отзыва:
- Пользователь должен иметь **хотя бы один пресет этого филамента** в профиле (saved_presets)
- ИЛИ должен быть **официальный пресет** филамента (тогда можно оставить отзыв без сохранения пресета)

### 2. Выбор пресета для отзыва:

**Сценарий A: У пользователя есть сохраненные пресеты этого филамента**
- При создании отзыва можно выбрать конкретный пресет из сохраненных
- Если не выбран → автоматически выбирается официальный пресет (если есть)
- Если официального нет → выбирается первый сохраненный пресет

**Сценарий B: У пользователя НЕТ сохраненных пресетов, но есть официальный пресет**
- Отзыв автоматически привязывается к официальному пресету
- Можно оставить отзыв на официальный пресет без сохранения его в профиль

**Сценарий C: Нет сохраненных пресетов И нет официального пресета**
- ❌ Нельзя оставить отзыв (показать сообщение: "Добавьте хотя бы один пресет этого филамента в профиль")

### 3. Архитектура:

```
FilamentReview
├── filament_id (обязательно)
├── preset_id (обязательно, но может быть NULL для отзывов "о филаменте в целом")
├── user_id (обязательно)
├── success (bool)
├── rating (1-5)
└── comment
```

**Логика:**
- Если `preset_id` указан → отзыв влияет на рейтинг этого пресета
- Если `preset_id = NULL` → отзыв влияет только на общий рейтинг филамента (но такие отзывы не должны быть в нормальной работе)

### 4. Вычисление рейтингов:

**Рейтинг пресета:**
- `preset.rating` = средний рейтинг всех отзывов с `preset_id = preset.id`
- `preset.success_rate` = процент успешных печатей по этому пресету
- Обновляется автоматически при создании/обновлении отзыва

**Рейтинг филамента:**
- `filament.rating` = взвешенное среднее рейтингов пресетов:
  ```
  filament_rating = Σ(preset.rating × preset.usage_count × weight) / Σ(preset.usage_count × weight)
  ```
  где `weight = 1.5` для официальных пресетов, `1.0` для пользовательских
- ИЛИ простое среднее всех отзывов:
  ```
  filament_rating = Σ(review.rating) / count(reviews)
  ```
- `filament.success_rate` = процент успешных печатей по всем отзывам

### 5. UI для создания отзыва:

**Форма создания отзыва:**
1. Выбор пресета (выпадающий список):
   - Показываются только сохраненные пресеты этого филамента
   - Если есть официальный пресет → добавляется в список с пометкой "Официальный"
   - Опция "Автоматически (официальный пресет)" по умолчанию
   
2. Рейтинг (1-5 звезд)
3. Успешность печати (да/нет)
4. Комментарий (опционально)
5. Модель принтера (опционально)

### 6. Валидация при создании отзыва:

```python
# Проверка при создании отзыва:
1. Проверить, что filament_id существует
2. Если preset_id указан:
   - Проверить, что пресет существует
   - Проверить, что preset.filament_id == filament_id
   - Проверить, что пресет либо сохранен пользователем, либо официальный
3. Если preset_id не указан:
   - Найти официальный пресет филамента
   - Если есть → установить preset_id = official_preset.id
   - Если нет → ошибка "Выберите пресет для отзыва"
```

### 7. Изменения в БД:

```sql
-- Добавить preset_id в FilamentReview (опциональное поле)
ALTER TABLE filament_reviews ADD COLUMN preset_id INTEGER REFERENCES presets(id);
CREATE INDEX ix_filament_reviews_preset_id ON filament_reviews(preset_id);

-- Добавить success_rate в Preset (вычисляемое или хранимое)
ALTER TABLE presets ADD COLUMN success_rate FLOAT;
```

### 8. API изменения:

**Создание отзыва:**
```python
POST /api/v1/filament-reviews/
{
  "filament_id": 1,
  "preset_id": 5,  # опционально, если не указан - используется официальный
  "success": true,
  "rating": 4.5,
  "comment": "...",
  "printer_model": "..."
}
```

**Получение пресетов для выбора:**
```python
GET /api/v1/filaments/{filament_id}/available-presets-for-review
# Возвращает список пресетов, на которые можно оставить отзыв:
# - сохраненные пользователем пресеты этого филамента
# - официальный пресет (если есть)
```

### 9. Обновление рейтингов пресетов:

При создании/обновлении/удалении отзыва:
1. Пересчитать `preset.rating` из всех отзывов этого пресета
2. Пересчитать `preset.success_rate` из всех отзывов этого пресета
3. Обновить `preset.updated_at` для инвалидации кэша

### 10. Примеры:

**Пример 1:**
- Филамент: PLA от Brand X
- У пользователя сохранены: пресет A (community), пресет B (community)
- Есть официальный пресет: Official
- При создании отзыва можно выбрать: A, B, или Official (по умолчанию)

**Пример 2:**
- Филамент: PETG от Brand Y
- У пользователя нет сохраненных пресетов
- Есть официальный пресет: Official
- При создании отзыва автоматически выбирается Official

**Пример 3:**
- Филамент: ABS от Brand Z
- У пользователя нет сохраненных пресетов
- Нет официального пресета
- ❌ Нельзя оставить отзыв (показать сообщение)
