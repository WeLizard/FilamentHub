# Сценарии использования профилей на FilamentHub

## 👥 Типы пользователей

### 1. Опытный 3D печатник
- Умеет создавать крутые профили
- Может создавать все типы профилей:
  - PrinterProfile (настройки принтера + сопло)
  - PrintProfile (профили печати)
  - Preset (настройки для филамента)
- Знает, что и как настраивать

### 2. Новичок
- Только купил 3D принтер
- Не разбирается в настройках
- Знает только название своего принтера (например, "Ender 3 Pro")
- Нужна помощь в выборе настроек

### 3. Производитель филамента
- Загружает свой филамент на сайт
- Может создавать пресеты для своего филамента
- Хочет, чтобы другие пользователи могли легко печатать его материалом

---

## 🎯 Сценарии использования

### Сценарий 1: Новичок хочет напечатать филаментом

**Проблема:** У новичка есть принтер и филамент, но он не знает, какие настройки использовать.

**Решение:**
1. Новичок открывает страницу филамента (например, "PLA Red" от Sunlu)
2. Видит список пресетов для этого филамента (официальные, взвешенные, пользовательские)
3. Выбирает понравившийся пресет (например, "Официальный от Sunlu")
4. Система предлагает:
   - "Рекомендуемые профили печати" (PrintProfile)
   - "Рекомендуемые профили принтера" (PrinterProfile) - если новичок указал свой принтер в профиле
5. Новичок выбирает всё по рекомендациям
6. Нажимает "Экспорт в OrcaSlicer" → получает готовую комбинацию

**Что важно:**
- Новичок не должен создавать профили - он только выбирает из готовых
- Система должна автоматически подбирать совместимые варианты
- Профили принтера и печати могут быть просто "рекомендациями", а не обязательными

---

### Сценарий 2: Опытный печатник создаёт свои профили

**Проблема:** Опытный печатник хочет сохранить свои настройки и поделиться ими.

**Решение:**
1. Опытный печатник создаёт:
   - PrinterProfile для своего принтера
   - PrintProfile для своих настроек печати
   - Preset для конкретного филамента
2. Сохраняет их на сайте (чтобы не потерять)
3. Может экспортировать всё сразу в OrcaSlicer
4. Может поделиться с другими пользователями

**Что важно:**
- Опытный печатник может создавать всё сам
- Профили могут быть "личными" (только для хранения) или "публичными" (для других)
- Можно создать полную комбинацию и сохранить её

---

### Сценарий 3: Производитель филамента загружает материал

**Проблема:** Производитель хочет, чтобы его материалом было легко печатать.

**Решение:**
1. Производитель загружает филамент (например, "PLA Pro Red")
2. Создаёт официальные пресеты для этого филамента:
   - Для разных принтеров (Ender 3, Prusa MK4, Voron, etc.)
   - Для разных качеств (Standard, Fine, Draft)
3. Система автоматически связывает пресеты с принтерами через `preset_printers`
4. Другие пользователи могут:
   - Выбрать филамент
   - Выбрать пресет от производителя
   - Получить рекомендации по профилям печати и принтера

**Что важно:**
- Производитель создаёт только филамент и пресеты
- Профили печати и принтера могут быть от других пользователей
- Система автоматически подбирает совместимые профили

---

## 🏗️ Архитектура решения

### Вариант 1: Опциональная связь (рекомендуется)

**Идея:** Filament + Preset - это **основа**, а PrinterProfile и PrintProfile - это **опциональные рекомендации**.

**Логика:**
1. Пользователь **обязательно** выбирает:
   - Filament (материал)
   - Preset (настройки для материала)
2. Пользователь **опционально** выбирает:
   - PrintProfile (профиль печати) - можно не выбирать, использовать дефолтный из OrcaSlicer
   - PrinterProfile (профиль принтера) - можно не выбирать, использовать дефолтный из OrcaSlicer

**Преимущества:**
- Простота для новичков: они выбирают только филамент и пресет
- Гибкость для опытных: они могут выбрать всё
- Профили принтера и печати могут быть просто "для хранения" на сайте

**Недостатки:**
- Новичок может не знать, какой PrintProfile выбрать
- Нужны умные рекомендации

---

### Вариант 2: Автоматический подбор

**Идея:** Система автоматически подбирает PrintProfile и PrinterProfile на основе выбранного Preset.

**Логика:**
1. Пользователь выбирает:
   - Filament + Preset
2. Система автоматически находит:
   - PrintProfile, который совместим с:
     - Принтером пользователя (если указан в профиле)
     - Filament через `print_profile_filaments`
   - PrinterProfile, который совместим с:
     - Принтером пользователя
     - PrintProfile через `default_print_profile_slug`

**Преимущества:**
- Минимум действий для новичка
- Автоматический подбор лучших вариантов
- Можно переопределить автоматический выбор

**Недостатки:**
- Нужен хороший алгоритм подбора
- Может быть несколько вариантов - какой выбрать?

---

### Вариант 3: Уровни детализации

**Идея:** Разные уровни детализации для разных пользователей.

**Уровень 1: Минимальный (для новичков)**
- Filament + Preset → экспорт в OrcaSlicer
- OrcaSlicer сам подберёт профили принтера и печати

**Уровень 2: Средний (для продвинутых)**
- Filament + Preset + PrintProfile → экспорт
- Пользователь выбирает профиль печати, но не профиль принтера

**Уровень 3: Полный (для опытных)**
- Filament + Preset + PrintProfile + PrinterProfile → экспорт
- Пользователь выбирает всё сам

**Преимущества:**
- Каждый пользователь выбирает нужный ему уровень
- Новички не путаются в деталях
- Опытные могут настроить всё до мелочей

**Недостатки:**
- Нужно объяснить пользователю, что такое каждый уровень
- Может быть неочевидно, какой уровень выбрать

---

## 💡 Предлагаемое решение: Гибридный подход

### Принципы:

1. **Filament + Preset - это основа**
   - Без них нельзя начать печать
   - Это главный выбор пользователя

2. **PrintProfile и PrinterProfile - опциональные рекомендации**
   - Можно использовать для улучшения качества печати
   - Можно хранить на сайте просто "на будущее"
   - Можно не использовать вообще (OrcaSlicer подберёт сам)

3. **Умные рекомендации**
   - Если пользователь выбрал Filament + Preset → показать рекомендуемые PrintProfile и PrinterProfile
   - Если пользователь указал свой принтер в профиле → показать профили для этого принтера
   - Если пользователь опытный → дать возможность выбрать всё самому

## 🔄 Алгоритм работы

### Минимальный сценарий (для новичков):

1. Пользователь выбирает **Filament** → **Preset**
2. Нажимает **"Экспорт в OrcaSlicer"**
3. Система экспортирует только Filament + Preset
4. OrcaSlicer сам подберёт PrintProfile и PrinterProfile (из своих дефолтных)

### Расширенный сценарий (для опытных):

1. Пользователь выбирает **Filament** → **Preset**
2. Система показывает **рекомендуемые PrintProfile и PrinterProfile**
3. Пользователь выбирает (или пропускает)
4. Нажимает **"Экспорт в OrcaSlicer"**
5. Система экспортирует всю выбранную комбинацию

### Полный сценарий (для экспертов):

1. Пользователь создаёт **PrinterProfile** для своего принтера
2. Пользователь создаёт **PrintProfile** для своих настроек
3. Пользователь выбирает **Filament** → **Preset**
4. Пользователь связывает всё вместе
5. Сохраняет комбинацию на сайте
6. Экспортирует в OrcaSlicer

## ✅ Выводы

### Главные принципы:

1. **Filament + Preset - это главное**
   - Это минимум, что нужно для печати
   - Все пользователи (новички и опытные) используют это

2. **PrintProfile и PrinterProfile - опциональные улучшения**
   - Можно использовать, но не обязательно
   - Могут быть просто "для хранения" на сайте
   - OrcaSlicer может подобрать их автоматически

3. **Умные рекомендации**
   - Если пользователь выбрал Filament + Preset → показать рекомендуемые профили
   - Если пользователь указал свой принтер → показать профили для этого принтера
   - Но не навязывать - пусть пользователь сам решает

4. **Разные уровни детализации**
   - Новичок: выбирает только Filament + Preset
   - Продвинутый: выбирает Filament + Preset + PrintProfile (опционально)
   - Опытный: выбирает всё сам, создаёт свои профили

## 📝 Что нужно реализовать

### На странице филамента:

1. **Блок выбора пресета** (обязательно)
   - Список всех пресетов для филамента
   - Фильтры: официальные, взвешенные, пользовательские
   - Кнопка "Выбрать"

2. **Блок рекомендуемых профилей** (опционально)
   - Показывается только если пользователь выбрал пресет
   - Рекомендуемые PrintProfile (если есть совместимые)
   - Рекомендуемые PrinterProfile (если есть совместимые и если пользователь указал свой принтер)
   - Текст: "Можете не выбирать - OrcaSlicer подберёт автоматически"

3. **Блок экспорта**
   - Кнопка "Экспорт в OrcaSlicer"
   - Показывает, что будет экспортировано
   - Экспортирует только выбранные элементы

### В профиле пользователя:

1. **"Мои пресеты"** (для всех)
   - Список сохранённых пресетов (Filament + Preset)
   - Кнопка "Экспорт"

2. **"Мои комбинации"** (для опытных)
   - Полные комбинации: PrinterProfile + PrintProfile + Filament + Preset
   - Показывается только если пользователь создал PrinterProfile или PrintProfile
   - Кнопка "Экспорт" для всей комбинации
