# FilamentHub — Гайд для AI-агентов (Playwright)

> Этот документ описывает структуру сайта FilamentHub и пошаговые инструкции для AI-агентов (Gemini, Qwen, Claude) работающих через Playwright MCP.
> Также может использоваться как пользовательская документация по разделу «Профиль».

## Окружение

- **Dev URL**: `http://localhost:3000`
- **Backend API**: `http://localhost:3000/api/v1/...` (проксируется через Vite/nginx)
- **Swagger**: `http://localhost:8001/api/v1/docs`
- **Язык интерфейса**: русский (ru-RU) по умолчанию

## Тестовые аккаунты

| Роль | Логин | Пароль | Описание |
|------|-------|--------|----------|
| Админ | `admin` | `admin123` | Полный доступ, админ-панель |
| Пользователь | `test@test.com` | `Test123456` | Обычный юзер |

Для создания нового пользователя — см. раздел «Регистрация».

## Роли пользователей

| Роль | Что может |
|------|-----------|
| **user** | Создавать бренды, филаменты (для неверифицированных брендов), пресеты, профили принтеров. Сохранять чужие пресеты. Оставлять отзывы. |
| **brand** | Всё что user + управление своим брендом, создание официальных пресетов, филаментов для своего бренда |
| **admin** | Всё + админ-панель, модерация, управление пользователями |

## Авторизация

### Вход (логин)

1. Нажать кнопку **«Войти»** в верхнем меню
2. Откроется модалка с формой
3. Заполнить поля:
   - **«Эл. почта или логин»** — email или username
   - **«Пароль»** — пароль
4. Нажать кнопку **«Войти»** внутри формы
5. Модалка закроется, в меню появятся «Профиль» и «Выйти»

```
Пример (Playwright):
1. click "Войти" (кнопка в навбаре)
2. fill "эл. почта или логин" → "test@test.com"
3. fill "••••••••" → "Test123456"
4. click "Войти" (кнопка в форме — внутри <form>)
```

**ВАЖНО**: Кнопка «Войти» есть и в навбаре, и в форме. Для отправки нужно кликать кнопку **внутри формы**. В Playwright: `page.locator('form').getByRole('button', { name: 'Войти' }).click()`

### Регистрация

1. Нажать **«Войти»** → в модалке нажать вкладку **«Регистрация»**
2. Заполнить:
   - **Имя пользователя** (username, уникальный)
   - **Эл. почта**
   - **Пароль** (мин. 8 символов, буквы + цифры)
   - **Подтверждение пароля**
3. Нажать **«Зарегистрироваться»**

### Выход

- Нажать кнопку **«Выйти»** в верхнем меню

### Вкладка «Пресеты»

Управление пресетами — создание, редактирование, удаление, скачивание.

**Заголовок**: «Профили филамента» + кнопка **«Новый пресет»**

**Список пресетов** — карточки с подробной информацией:

Каждая карточка содержит:
- **Название пресета** (заголовок, кликабельный)
- **Филамент**: бренд • название • цвет • тип материала (кликабельная ссылка на страницу бренда)
- **Кнопки действий** (справа от заголовка):
  - ✏️ **Редактировать** — открывает модалку редактирования (только для своих пресетов)
  - 🔄 **Синхронизация** — вкл/выкл синхронизации с OrcaSlicer
  - ⬇️ **Скачать** — скачать в формате OrcaSlicer (.json)
  - 🗑️ **Удалить** — удалить пресет или убрать сохранённый из профиля
- **Параметры** (сетка):
  - Сопло: 200°C
  - Стол: 60°C
  - Скорость: 50mm/s
  - Перемещение: 150mm/s
  - Поток: 100%
  - Обдув: 100%
  - Ретракт: 5mm
  - Ск. ретракт: 45mm/s
  - Использований: 0
- **Дата создания** внизу

```
Playwright — действия с пресетом:
- click "Редактировать" (title="Редактировать")
- click "Скачать в формате OrcaSlicer"
- click "Удалить" (title="Удалить")
- click "Синхронизация включена. Нажмите, чтобы отключить."
```

#### Создание пресета (подробный flow)

1. На вкладке «Пресеты» нажать **«Новый пресет»**
2. Откроется модалка **«Создать новый пресет»** (метка «Формы Orca» вверху)

**Шаг 1 — Выбор филамента:**
- Поле **«Филамент»** — автокомплит, начать вводить название
- Из выпадающего списка выбрать существующий филамент
- Или нажать кнопку **«+»** (плюсик) справа от поля → откроется инлайн-форма

**Шаг 1а — Создание нового филамента (если нужен):**
- Поле **«Производитель»** — автокомплит по существующим брендам. Если бренда нет — ввести название нового
- **«Название филамента»** — например: `PLA+ Mate R24`
- **«Тип материала»** — выбрать из списка (PLA, ABS, PETG, TPU, PA, PC, ASA, HIPS, PVA...) или ввести кастомный
- **«Цвет»** — клик по цветовому кружку, выбрать цвет
- Дополнительные поля (опционально): диаметр, плотность, описание
- Нажать кнопку подтверждения

**Шаг 2 — Параметры пресета:**
- **Название пресета** (обязательно)
- **Температура сопла (°C)** — обязательно
- **Температура стола (°C)** — обязательно
- **Скорость печати (mm/s)**
- **Скорость перемещений (mm/s)**
- **Поток (%)**
- **Скорость вентилятора (%)**
- **Длина ретракта (mm)**
- **Скорость ретракта (mm/s)**
- **Принтеры** (опционально) — можно привязать к одному или нескольким принтерам из базы

**Шаг 3 (опционально) — Подробные настройки:**

Раскрыть блок **«Подробные настройки»** → появятся вкладки:

| Вкладка | Содержимое |
|---------|-----------|
| **Профиль прутка** | Усадка XY/Z, температура размягчения, Pressure Advance, объёмный расход, адаптивная объёмная скорость, длина набивки филамента |
| **Охлаждение** | Управление вентилятором по слоям, пороги скорости, обдув нависаний и мостов, скорость вентилятора при глажке |
| **Переопределение параметров** | Переопределение скоростей, ускорений, ширины линий |
| **Дополнительно** | Дополнительные OrcaSlicer-параметры |
| **Экструдер мм** | Настройки мультиматериальной системы |
| **Заметки** | Произвольный текст |

**Шаг 4 — Создание:**
- Нажать **«Создать»**
- Пресет появится в списке на вкладке «Пресеты»

```
Полный пример (Playwright):
1. navigate "http://localhost:3000/profile"
2. click "Пресеты"
3. click "Новый пресет"
4. --- Выбор филамента ---
   click "+" (кнопка плюс рядом с полем филамента)
   fill "Производитель" → "Bestfilament"
   fill "Название филамента" → "PLA Pro"
   click выбрать тип "PLA"
   click выбрать цвет
5. --- Параметры пресета ---
   fill "Название пресета" → "BF PLA Pro 210/60"
   fill "Температура сопла" → "210"
   fill "Температура стола" → "60"
   fill "Скорость печати" → "100"
   fill "Скорость перемещений" → "150"
   fill "Поток" → "98"
   fill "Скорость вентилятора" → "100"
   fill "Длина ретракта" → "0.8"
   fill "Скорость ретракта" → "40"
6. click "Создать"
```

### Вкладка «Калькулятор»

Калькулятор стоимости печати с учётом региональных особенностей (Россия).

**Метод расчёта** (переключатель):
- **По граммам** — расчёт по весу материала
- **По часам** — расчёт по времени печати
- **Комбинированный** — оба метода вместе

**Секции формы:**

| Секция | Поля |
|--------|------|
| **Параметры материала** | Вес детали (г), вес поддержек (г), коэф. потерь, цена катушки (₽), вес катушки (кг), доставка (₽) |
| **Время печати** | Часы, минуты, секунды |
| **Электроэнергия** | Стоимость 1 кВт·ч (₽), мощность принтера (Вт) |
| **Дополнительные услуги** | Моделирование (ч/мин + ставка), ставка печати (₽/ч), постобработка (ч/мин + ставка), амортизация (₽/ч) |
| **Накладные расходы** | Накладные (%), наценка (%) |
| **Коэффициенты** | Срочность, сложность, скидка за объём |
| **Итого** | Количество деталей, фиксированные расходы, мин. цена заказа, округление |

Кнопка **«Рассчитать»** → показывает детализированный расчёт стоимости.

```
Playwright — быстрый расчёт:
1. click "Калькулятор"
2. fill "Вес детали" → "100"
3. fill "Часы" → "2"
4. fill "Минуты" → "30"
5. click "Рассчитать"
```

### Профиль компании (режим «Компания»)

Доступен по нажатию кнопки **«Компания»** / **«Профиль компании»** вверху страницы профиля.

Показывает информацию о верификации и форму подачи заявки.

**Если пользователь ещё не представитель бренда:**

Заголовок: **«Получите доступ к кабинету производителя»**

Описание процесса верификации (4 шага):
1. Выбрать: присоединиться к существующему бренду или зарегистрировать новый
2. Указать подтверждающие документы
3. Администрация проверит (3–5 рабочих дней)
4. После одобрения — доступ к кабинету производителя

**Два варианта подачи заявки:**

**Вариант 1: Присоединиться к существующему бренду**
- Поле автокомплита: «Начните вводить название бренда...»
- Выбрать бренд из списка → подать заявку

**Вариант 2: Зарегистрировать новый бренд**
- Кнопка **«Зарегистрировать новый бренд»**
- Откроется форма: название, сайт, описание, контакты
- Подать заявку на модерацию

```
Playwright — подача заявки на существующий бренд:
1. navigate "/profile"
2. click "Компания"
3. fill "Начните вводить название бренда..." → "Bestfilament"
4. Выбрать из dropdown
5. Заполнить обоснование
6. click "Подать заявку"

Playwright — регистрация нового бренда:
1. navigate "/profile"
2. click "Компания"
3. click "Зарегистрировать новый бренд"
4. fill название, сайт, описание
5. click "Подать заявку"
```

1. Перейти в **Админка** → вкладка **«Принтеры»**
2. Нажать **«Добавить принтер»**
3. Откроется модалка с вкладками:
   - **Общая информация** — название*, производитель*, модель*, slug*, размеры, материалы, медиа, конфигурация Orca
   - **Кинематика** — лимиты скоростей (мм/с), ускорений (мм/с²), jerk
   - **Экструдеры** — диаметр сопла, макс. температуры, дополнительные сопла
   - **Мультиматериал** — single-extruder MM настройки
   - **G-code принтера** — стартовый/финальный G-code, пауза, таймлапс
   - **Заметки** — описание и служебные комментарии
4. Заполнить обязательные поля (отмечены `*`):
   - **Название** (например: `Bambu Lab X1 Carbon`)
   - **Производитель** (например: `Bambu Lab`)
   - **Модель** (например: `X1 Carbon`)
   - **Slug** — нажать **«Генерировать»** для автогенерации
5. Нажать **«Создать»**

```
Playwright:
1. navigate "/admin"
2. click "Принтеры"
3. click "Добавить принтер"
4. fill "Название" → "Bambu Lab X1 Carbon"
5. fill "Производитель" → "Bambu Lab"
6. fill "Модель" → "X1 Carbon"
7. click "Генерировать" (для slug)
8. fill "Ширина X (мм)" → "256"
9. fill "Глубина Y (мм)" → "256"
10. fill "Высота Z (мм)" → "256"
11. click "Создать"
```

### Одобрение заявки на верификацию бренда

1. Перейти в **Админка** → вкладка **«Заявки на верификацию»**
2. Фильтры: Все / Ожидают / Одобрены / Отклонены
3. Найти заявку со статусом «Ожидает»
4. Нажать **«Одобрить»** или **«Отклонить»**

## API (для прямых вызовов)

Если Playwright-навигация слишком медленная, можно использовать API напрямую.

### Авторизация
```
POST /api/v1/auth/login
Content-Type: application/x-www-form-urlencoded
Body: username=test@test.com&password=Test123456

Response: { "access_token": "...", "refresh_token": "...", "token_type": "bearer" }

Далее: Header "Authorization: Bearer {access_token}"
```

### Регистрация
```
POST /api/v1/auth/register
Body: {
  "email": "user@example.com",
  "username": "myuser",
  "password": "MyPassword123"
}
```

### Создание бренда
```
POST /api/v1/brands/
Body: { "name": "My Brand", "website": "https://mybrand.com" }
```

### Создание филамента
```
POST /api/v1/filaments/
Body: {
  "name": "PLA Standard",
  "brand_id": 1,
  "material_type": "PLA",
  "color_name": "Red",
  "color_hex": "#FF0000",
  "diameter": 1.75,
  "density": 1.24
}
```

### Создание пресета
```
POST /api/v1/presets/
Body: {
  "name": "PLA Standard 210/60",
  "filament_id": 1,
  "extruder_temp": 210,
  "bed_temp": 60,
  "print_speed": 100,
  "travel_speed": 150,
  "flow_rate": 100,
  "fan_speed": 100,
  "retraction_length": 0.8,
  "retraction_speed": 40
}
```

### Сохранение пресета в профиль
```
POST /api/v1/saved-presets/
Body: { "preset_id": 1 }
```

### Полезные GET-запросы
```
GET /api/v1/auth/me                         — текущий пользователь
GET /api/v1/brands/?page=1&size=100         — список брендов
GET /api/v1/filaments/?page=1&size=100      — список филаментов
GET /api/v1/presets/?page=1&size=100        — список пресетов
GET /api/v1/printers/?page=1&size=100       — список принтеров
GET /api/v1/saved-presets/                  — сохранённые пресеты
GET /api/v1/notifications/?page=1&size=50   — уведомления
```

## Типичные тестовые сценарии

### Сценарий 1: Обычный пользователь создаёт пресет

1. Зарегистрироваться или войти как `test@test.com` / `Test123456`
2. Перейти в Профиль → Пресеты → Новый пресет
3. Нажать «+» рядом с полем филамента
4. Создать бренд (ввести название) → создать филамент (название, тип, цвет)
5. Заполнить параметры пресета (температуры, скорости, ретракт)
6. Нажать «Создать»
7. Проверить что пресет появился в списке на вкладке «Пресеты»

### Сценарий 2: Пользователь сохраняет чужой пресет

1. Войти
2. На главной странице (Каталог) найти карточку с пресетом
3. Нажать **«+В профиль»**
4. Перейти в Профиль → Пресеты
5. Проверить что пресет появился

### Сценарий 3: Представитель бренда (полный цикл)

1. Зарегистрировать нового пользователя (через UI или API)
2. Войти → Профиль → нажать **«Компания»**
3. **Вариант 1**: Ввести имя существующего бренда → выбрать из autocomplete → подать заявку
4. **Вариант 2**: Нажать «Зарегистрировать новый бренд» → заполнить форму → подать заявку
5. Выйти → Войти как админ (`admin` / `admin123`)
6. Перейти в Админка → Заявки на верификацию → Одобрить заявку
7. Выйти → Войти обратно как пользователь-представитель
8. Теперь у пользователя роль `brand` и он может:
   - Создавать филаменты для своего бренда
   - Создавать официальные пресеты (чекбокс «Официальный пресет» в форме создания)
   - Управлять профилем бренда

### Сценарий 4: Админ создаёт принтер

1. Войти как `admin` / `admin123`
2. Перейти в Админка → Принтеры → Добавить принтер
3. Заполнить: Название, Производитель, Модель
4. Нажать «Генерировать» для Slug
5. Заполнить размеры (X, Y, Z)
6. Нажать «Создать»

### Сценарий 5: Массовое создание контента (для наполнения базы)

Рекомендуется использовать **API напрямую** (см. раздел API):

1. Зарегистрировать пользователя через `POST /api/v1/auth/register`
2. Залогиниться через `POST /api/v1/auth/login`, получить `access_token`
3. Создать 5-10 брендов через `POST /api/v1/brands/`
4. Для каждого бренда создать 3-5 филаментов через `POST /api/v1/filaments/`
5. Для каждого филамента создать 1-3 пресета через `POST /api/v1/presets/`

### Сценарий 6: Расчёт стоимости печати

1. Войти → Профиль → Калькулятор
2. Ввести вес детали (г), время печати (ч:мин)
3. Проверить/изменить цену катушки, стоимость электроэнергии
4. Задать наценку и накладные расходы
5. Нажать «Рассчитать»
6. Проверить результат — детализированная стоимость

## Особенности и подводные камни

1. **Модалка логина** — кнопка «Войти» есть и в навбаре, и в форме. Для отправки формы нужно кликать кнопку **внутри формы**. В Playwright: `page.locator('form').getByRole('button', { name: 'Войти' }).click()`

2. **Автокомплит филамента** — при создании пресета поле филамента — это autocomplete. Нужно ввести текст и подождать dropdown, либо использовать кнопку «+» для создания нового.

3. **Slug принтера** — обязательное поле, но можно нажать «Генерировать» чтобы создать автоматически из производителя + модели.

4. **printer_ids в пресете** — необязательное поле (default: пустой список). Пресет можно создать без привязки к принтеру.

5. **Верификация бренда** — неверифицированные бренды может создать любой пользователь. Любой пользователь может создавать филаменты для неверифицированных брендов. Для верифицированных — только сотрудник бренда или админ.

6. **Официальные пресеты** — при создании пресета есть чекбокс «Официальный пресет (от производителя)». Он доступен только для сотрудников бренда (role=brand) и только для филаментов их бренда.

7. **Формы Orca** — метка в модалках означает что форма повторяет структуру OrcaSlicer.

8. **Переключатель Пользователь/Компания** — находится вверху страницы профиля. Это не вкладки, а именно переключатель режима — меняет весь контент страницы.

9. **API login** — endpoint использует `application/x-www-form-urlencoded`, а не JSON. Поле называется `username` (но принимает и email).

10. **Docker перезагрузка** — после изменения кода в `frontend/src/` Vite hot-reload подхватывает изменения автоматически. Для backend нужен рестарт контейнера: `docker restart filamenthub_backend_dev`.
