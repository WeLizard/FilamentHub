### Статьи по материалам (специфичные)

#### PLA свойства
- **Поиск:** "PLA mechanical properties 3D printing", "polylactic acid thermal properties"
- **Ключевые темы:** Механические свойства, температура стеклования, усадка

#### PETG свойства
- **Поиск:** "PETG filament properties", "polyethylene terephthalate glycol"
- **Ключевые темы:** Прозрачность, химическая стойкость, адгезия слоёв

#### ABS свойства
- **Поиск:** "ABS 3D printing properties", "acrylonitrile butadiene styrene"
- **Ключевые темы:** Ударная вязкость, температура деформации, запах

#### Nylon/PA свойства
- **Поиск:** "Nylon PA filament properties", "polyamide 3D printing"
- **Ключевые темы:** Гигроскопичность, прочность, температура плавления

#### TPU/Flex свойства
- **Поиск:** "TPU flexible filament properties", "thermoplastic polyurethane"
- **Ключевые темы:** Эластичность, Shore hardness, экструзия

## 📝 Как использовать этот индекс

### Для разработчиков FilamentHub:
1. **Книги** → Используй для понимания свойств материалов, температур, проблем печати
2. **Статьи** → Источник актуальных данных о свойствах материалов (механические, термические)
3. **Сайты** → Референсы для данных о филаментах, проблемах, решениях

### Для Wiki контента:
- Книги → Разделы "Материалы", "Проблемы печати", "Настройки"
- Статьи → Точные данные о свойствах (таблицы, графики)
- Сайты → Ссылки в статьях Wiki на внешние ресурсы

### Автоматизация:
- Используй `scripts/download_books.py` для скачивания книг
- Парсинг статей можно добавить позже для автоматического сбора данных
