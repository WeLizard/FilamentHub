## 🚀 Использование

### Скачать книги:

```bash
cd docs/resources
python scripts/download_books.py
```

Скрипт автоматически:
- Проверяет существующие файлы (не скачивает повторно)
- Показывает прогресс загрузки
- Сохраняет в папку `books/`

### Добавить новую книгу:

1. Отредактируй `scripts/download_books.py`
2. Добавь запись в список `BOOKS`
3. Запусти скрипт

## 🎯 Для разработчиков FilamentHub

### Как использовать книги:

1. **Поиск свойств материалов:**
   - Температуры плавления, стеклования
   - Механические свойства (прочность, модуль упругости)
   - Гигроскопичность, усадка

2. **Решение проблем печати:**
   - Таблицы проблем и решений
   - Настройки температуры, скорости
   - Постобработка

3. **Валидация данных:**
   - Проверка температур в пресетах против научных данных
   - Предупреждения о нестандартных настройках
